\chapter{Administración del Sistema}\label{chap-sysadm}
\markboth{Administración del Sistema}{}
\label{chap-sysadm-num}

Este capítulo cubre las cosas más importantes que se necesitan saber
acerca de la administración del sistema para comenzar a usarlo sin problemas. Para que el capítulo tenga un tamaño razonable,
cubre sólo lo básico y omite muchos detalles importantes. El libro
{\em Linux System Administrator's Guide}, de Lars Wirzenius (ver
Apéndice~\ref{app-info}) proporciona más detalles sobre la
administración del sistema. Además ayudará a entender mejor cómo
trabajan y cuelgan las cosas entre sí.
