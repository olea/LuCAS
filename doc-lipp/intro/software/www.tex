\subsection{World Wide Web}

Es necesario apuntar aquí que {\linux} lleva incluido el software de servidor
web y tambíen navegadores web.   Miles de sistemas Linux
ejecutan Apache en internet hoy en dia, entre ellos la sede de recursos de
Linux {\tt www.linuxresources.com}.

En las distribuciones de {\linux} se incluyen diferentes navegadores web, 
además de otros que pueden bajarse de internet. Entre los navegadores 
disponibles se cuentan Mosaic, Netscape, Arena, Amaya y Mozilla.

Linux proporciona soporte completo para Java, applets CGI y Perl, la 
herramienta estándar en el entorno de programación {\linux}.
