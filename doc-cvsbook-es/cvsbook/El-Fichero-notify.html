El Fichero notify

Este fichero se usa junto con las características de alarmas de CVS, que se describen en CVS avanzado. Nada de ello tendrá sentido hasta que comprenda qué son las alarmas (son una característica útil pero no esencial), así que vea CVS avanzado para obtener detalles sobre este fichero y sobre las alarmas.
