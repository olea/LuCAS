These files are:
1. TBXcsV02.xsd - The w3c schema version of the core structure.
2. TBXdsV1-0.xml - The xdr schema of the XCS file.
3. TBXDv04.xml - The XCS file (default list of datacategories in TBX)
4. TBXspecV1j.DOC - TBX spec
5. TBXcdv04.dtd -The DTD for the core structure
6. encodingGuidelines.htm - a file refered to in the spec that defines the contents of the XCS file
7. TBX_DataCategoriesV1.htm - a companion to the TBX_DataCategoriesV1.htm file that gives example encodings of every datcat. It should be put in the same directory as the TBX_DataCategoriesV1.htm file for the hyperlinking to work correctly.
