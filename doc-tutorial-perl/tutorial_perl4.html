4- Bibliografía

Learning Perl de Randall L. Schartz. O'Reilly & Associates, Inc. Abril 1994, USA.

Tal vez el libro mas amigable que jamas haya leído para aprender a programar.

Programming perl de Larry Wall y Randall L. Schartz, Oreilly & Associates, Inc. Marzo de 1992, USA

Como referencia y material de consulta de Perl4 es casi insuperable, además, es un verdadero gozo leer al autor de Perl.

Teach Yourself CGI Programming with Perl in a Week, de Herrman Eric; Ed. SAMS Net, 1996, USA

Para aquellos que prefieran el papel como fuente de informacion, estaran felices con este libro para dar un tratamiento completo a los CGI's con Perl 5.

Using CGI, Jeffry Dwight and Michael Erwin, QUE 1996, USA.

Para aquellos que deseen dominar el tema de los CGI's, con no solo un lenguaje para implementarlo, y seguramente extensas discuciones de los pros y contras de los diversos mecanismos de los CGI's, este es un libro capaz de satisfacerlos.
