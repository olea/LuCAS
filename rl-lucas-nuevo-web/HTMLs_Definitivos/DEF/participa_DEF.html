Coordinación

Listas de coordinación internas:

  • Para trabajos en documentación, lista LuCAS (normas de uso).
  • Para trabajos de coordinación y de infraestructura, lista LuCAS-Coord.
  • Para registrase como voluntario lista LuCAS-Voluntarios a donde sólo se enviarán solicitudes de voluntarios por parte de los coordinadores de proyectos.
  • Charla en tiempo real, para consultas: #lucas.
