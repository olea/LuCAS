Letras griegas

Existe un comando para cada letra mayúscula y minúscula del alfabeto griego. Generalmente el nombre del comando está definido como el nombre de la letra griega en inglés.

Tabla A-2. Ejemplo

TipoCódigo TeXSalida
Mayúscula

$ \Delta $

Minúscula

$ \delta $
