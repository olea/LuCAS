<h2 align=center>¿Qué es software libre?</h2>

Quien lo recibe puede:

<ul>
<li> usarlo como mejor le parezca, donde mejor le parezca.
<li> redistribuirlo a quien quiera, por los medios que quiera.
<li> modificarlo (y mejorarlo o adaptarlo).
<li> redistribuir las modificaciones
</ul>

Imprescindible: disponibilidad de código fuente.

<center>
<font size=+1>
<b>software libre &ne; software gratis</b>
</font>
</center>

	<li> Clon Unix
	<li> Núcleo Linux
    <li> Familia de aplicaciones GNU + BSD + XFree86 + otro software libre

	<h2> Deshaciendo mitos equivocados y falsas interpretaciones<p>
	<ul>
    <li> Linux no es un sistema operativo.
    <li> Linux no ha sido creado por aficionados.
    <li> Linux no es Unix.
    <li> Linux no es el único núcleo libre: FreeBSD, Hurd, etc.
    <li> Está documentado.
    <li> Sí se pueden compartir ficheros con usuarios de otros sistemas operativos
    <li> Usar Linux no impide usar nuestros ficheros antiguos.
	</Ul>
