@@GLO computadora no podrá mejorar la "esencia" de los datos ó información 
@@GLO con los que se trabaja. A lo sumo, podrá mantenerla.

@@GLO de Programación Gráfica (GPI) (Graphics Programming Interface (GPI),

@@ING Graphical User Interface, GUI
@@CAS Interfaz Gráfica para el Usuario
@@GLO Microsoft Windows es un ambiente mutitarea de Interfaz Gráfica para 
@@GLO el Usuario  (Graphical User Interface) (GUI) que funciona en 
@@GLO microcomputadoras basadas en D.O.S
@@GLO El ambiente brinda una interfaz estandar basada en menúes 
@@GLO desplegables, ventanas en pantalla, y un dispositivo indicador tal 
@@GLO como un "ratón" (ó mouse)
@@GLO A fin de que un programa pueda funcionar en el ambiente Windows, 
@@GLO debe estar específicamente diseñado para hacer un uso óptimo de estas 
@@GLO características.
@@GLO  
@@GLO Por lo general, GUI hace referencia a un sistema que permite que 
@@GLO varios programas se presenten en diferentes ventanas en pantalla. El 
@@GLO operador puede mover estas ventanas por toda la pantalla, modificar 
@@GLO sus tamaños, y pasar facilmente de un programa a otro. Los programas 
@@GLO poseen una interfaz consistente con el usuario, que hace uso de menúes 
@@GLO desplegables, ventanas de diálogo y objetos gráficos tales como 
@@GLO íconos, barras deslizantes (scroll bars) y "botones" (buttons).
@@GLO Como toda presentación en pantalla se encuentra en modo gráfico, 
@@GLO el texto puede presentarse en diferentes "fuentes" y tamaños, y puede 
@@GLO integrarse facilmente con datos gráficos. Además de un teclado, los 
@@GLO GUI generalmente soportan el uso de un ratón (mouse).
@@GLO Muchos de los conceptos del GUI  (Graphical User Interface  - 
@@GLO Interfaz Gráfica para el Usuario) fueron originalmente desarrollados 
@@GLO en el Centro de Investigación Palo Alto, de XEROX  (Xerox's Palo Alto 
@@GLO Research Center) (PARC), introduciéndoselos por primera vez en forma 
@@GLO exitosa en la microcomputadoras Apple MacIntosh.
@@GLO El programa Presentation Manager es una GUI para el OS/2. de IBM.
@@GLO

@@ING Graphics Interchange Format, GIF

@@GLO GIF es una abreviatura de Graphics Interchange Format (Formato de 
@@GLO Intercambio Gráfico). A principios de 1987, los ingenieros en gráficos 
@@GLO de Compuserve introdujeron este nuevo protocolo como un estandar para 
@@GLO intercambiar imagenes basadas en raster (imagen electrónica en bruto,
@@GLO sin ningún tipo de procesamiento) entre diversas computadoras.
@@GLO Algunos puntos intrigantes acerca de GIF son:
@@GLO Tiene en cuenta la tecnología futura al ser capaz de preservar 
@@GLO imagenes con una resolución de 16.000 por 16.000 pixeles ó puntos 
@@GLO individuales.
@@GLO Mientras que GIF puede manejar hasta 256 colores simultáneos, el 
@@GLO programa hace un uso óptimo de las propias capacidades de una 
@@GLO computadora.
@@GLO GIF usa un sofisticado método de compresión de datos para reducir 
@@GLO el tamaño del archivo hasta la mitad, ahorrando así tiempo en la 
@@GLO transferencia hacia la máquina del Usuario.
@@GLO

@@GLO sucesor del D.O.S. (que también fuera desarrollado por Microsoft para 
@@GLO IBM) aunque proyectos como Cairo ó Chicago parecen cuestionar dicha

@@GLO OS/2 usa el "modo protegido" es la operación de la CPU a fin de

@@GLO eficiente operación multitareas.

@@GLO responsable de mantener los archivos, administrar varios programas que 
@@GLO puedan estar ejecutándose simultáneamente y de permitir que estos 
@@GLO programas se comuniquen entre si.
@@GLO Como segundo componente fundamental del OS/2, el Administrador
@@GLO de Presentaciones (Presentation Manager) extiende las capacidades 
@@GLO multitareas del núcleo a fin de ejecutarlos simultáneamente en la 
@@GLO misma pantalla, ocupando cada programa su propia pantalla. La Interfaz 
@@GLO de Programación Gráfica (GPI) (Graphics Programming Interface (GPI), 
@@GLO como tercer componente fundamental del sistema OS/2 brinda al sistema 
@@GLO un potente sistema de gráficos.
