	Esta sección está dedicada al mantenimiento del proyecto de
	documentación en español de Squish Squid-doc

	Esta sección se mantiene vía CVS desde :pserver:TULOGIN@cvs.hispalinux.es:/home/cvs/lucas

	- se recomienda ofrecer al menos una versión html navegable, otra
	html comprimida con tgz, otra postscript comprimida con gz y
	otra con las fuentes del artículo comprimida con gz o tgz;

	- si la versión html ocupa más de un fichero, crea un nuevo
	subdirectorio, de nombre HTML, y asegúrate de que existe un
	index.html;

	- aconsejamos encarecidamente el uso de DocBook o Linuxdoc-SGML para
	la composición de los artículos;

	- si es vuestro deseo, colgando de este directorio puede estar la
	página web del proyecto que también podría ser automáticamente
	publicada a través del web de LuCAS.
