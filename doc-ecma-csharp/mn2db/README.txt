mn2db.cs
 A simple program to transform a MonoDoc document to DocBook format
 via XSL Transformation

 maintainer: Ricardo Varela - phobeo at ieee dot org
-------------------------------------------------------------------

mn2db is a program designed to help people who need to work with monodoc
format to easily transform their documents into DocBook, a standard
documentation format that can be converted to different formats,
such as HTML or PDF, in a very simple way. For more information about DocBook, 
you can visit http://www.docbook.org

mn2db was originally created as part of the ecma334 translation project of
the mono hispano community, in collaboration with LuCAS. The project home 
page can be visited at: 
http://wiki.hispalinux.es/moin/ECMA_5fC_23
