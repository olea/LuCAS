Página con sonido para Internet Explorer

Esta página tiene sonido, pero este sólo funciona con Internet Explorer al usar su extensión BGSOUND. El código usado es:
 
Código usado para insertar el sonido
<BGSOUND SRC="reggae.mid">

 
NOTA: Puede pararse el sonido pulsando el botón DETENER:
NOTA: Usando la etiqueta EMBED puede insertarse sonido para Netscape Navigator y otros navegadores (incluido Internet Explorer 4.0). Además se puede usar la etiqueta NOEMBED para incluir el código que hemos usado en este ejemplo. Hemos preparado otra página que muestra cómo se hace.
