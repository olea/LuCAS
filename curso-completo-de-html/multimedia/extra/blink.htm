EJEMPLOS DE BLINK

NOTA: Estos ejemplos deben ser vistos con Netscape Navigator porque el resto de navegadores no soportan la etiqueta BLINK.
Ejemplo 1: Texto normal parpadeante
<BLINK>Este texto parpadea</BLINK>
Este texto parpadea

 
Ejemplo 2: Encabezado H1 parpadeante
<H1>
<BLINK>Este encabezado parpadea</BLINK>
</H1>

Este encabezado parpadea


 
Ejemplo 3: Texto normal + una imagen
<BLINK>
Este texto parpadea
<IMG SRC="spot.gif" WIDTH=32 HEIGHT=32>
</BLINK>
Este texto parpadea
