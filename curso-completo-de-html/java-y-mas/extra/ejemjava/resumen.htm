RESUMEN DE LA ETIQUETA APPLET

La etiqueta APPLET fue creada para posibilitar la inclusión de applets en las páginas Web. A continuación mostramos todos sus atributos y subetiquetas junto con una breve descripción. Los atributos CODE, WIDTH y HEIGHT deben ser usados obligatoriamente. El resto de atributos así como la subetiqueta PARAM son optativos, para indicarlo se muestran entre corchetes. Estos corchetes no forman parte del códgigo.
 
< APPLET Etiqueta usada para insertar APPLETS
    CODE = applet  Indica el nombre del archivo que contiene el applet
    HEIGHT = alto  Especifica la altura en pixeles que ocupará el applet
    WIDTH = ancho Especifica la anchura en pixeles que ocupará el applet
    [CODEBASE = URL] Dirección URL o ruta de directorios donde se encuentra el archivo indicado en CODE
    [ALT = 'Texto alternativo'] Texto alternativo
    [NAME = 'Nombre'] Nombre del applet (función informativa y para poder identificarlo)
    [ALIGN = alineamiento]  Permite alinear el applet. Puede tomar tres valores: right, left y center
    [VSPACE = pixeles] Margen en pixeles arriba y abajo del applet
    [HSPACE = pixeles] Margen en pixeles a derecha e izquierda del applet
    [ARCHIVE = archivo_comprimido] (Sólo Netscape Navigator) Permite indicar un archivo comprimido que contenga todos los archivos auxiliares que necesita el applet.
    [MAYSCRIPT] (Extensión) Indica que el applet puede ser usado desde un lenguaje de Script
    [TITLE= "título"] (Sólo Internet Explorer 4.0) Permite dar un título al applet que será mostrado en un bocadillo al pasar el cursos por encima.
[< PARAM NAME = Parametro1 VALUE = valor1 >]
[< PARAM NAME = Parametro2 VALUE = valor1 >]
. . . 		Parámetros extras específicos de cada applet.
[Código HTML alternativo] Código alternativo para navegadores que no entiendan la etiqueta APPLET
</APPLET>
