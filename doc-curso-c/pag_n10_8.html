10.8.- Funciones fprintf() y fscanf()


Todas las funciones de entrada y salida de datos que hemos visto hasta ahora tienen su simil en la E/S de ficheros, con la diferencia que debemos de indicar al el puntero al fichero para poder leer en de él o escribir. Dicha similitud la podemos observar con fprintf() y fscanf() comparándolas con printf() y scanf().



Sinopsis fprintf()

#include <stdio.h>

int fprintf(FILE *stream, const char *format, ....);

Compatibilidad UNIX, MSDOS

Devuelve el número de caracteres escritos o un número negativo en caso de error



La función fprintf() escribe los argumentos especificados entre paréntesis en el fichero apuntado. Un ejemplo de como poder implementar esta función lo podemos ver a continuación. El bucle presentado se supone entresacado de un programa. Lo que haría este bucle , sería escribir en el fichero apuntado por stream la variable número durante 7 veces.

for (i=0; i<7; i++)

{

  printf("Escribe números enteros: ");

  scanf("%d", &numero);

  fprintf(stream, "%d\n", numero);

}



La función fscanf() hace lo mismo que su correspondiente scanf()



Sinopsis fscanf()

#include <stdio.h>
int fscanf(FILE *stream, const char *format, ....);

Compatibilidad con UNIX, MSDOS

Devuelve el número de datos asignados, si devuelve 0, significa que no ha habido asignación y devuelve EOF si ha llegado al fin del fichero



Un fragmento de un programa usando fscanf() sería:

for(i=0; i<7; i++)

{

  fscanf(stream, "%d", &numero);

  total += numero;

}



Este bucle asigna valores de tipo entero a la variable numero procedentes del fichero apuntado por stream, y al tiempo que va renovando su valor, es sumado a la variable int total.

Podemos sacar resultados por pantalla con la función fprintf() haciendo que el fichero al que apunte fprintf() sea stdout:


fprintf(stdout, "fprintf() %s", "es una función de salida de datos");


Esta línea imprimiría por pantalla

fprintf() es una función de salida de datos


Al mismo tiempo tambien podemos leer datos del teclado (stdin) con la función fscanf():

fscanf (stdin, "%d", entero);


Es de señalar que esta no es una adecuada manera de guardar datos en un fichero, pues si bien necesita un byte por cada carácter que se almacena, para almacenar datos de tipo int necesita 1 bye por cada digito empleado en el número. Por ejemplo el número 12345678 necesita 8 bytes con el consiguiente desperdicio de memoria.
