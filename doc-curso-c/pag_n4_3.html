4.3.- Función scanf()                                           




#include <stdio.h>
int scanf()

Es compatible con ANSI, UNIX  y MSDOS


Esta maravillosa función permite al programa comunicarse con el exterior comunicarse con el exterior . scanf()se mantiene expectante captando datos del teclado hasta que pulsemos la tecla <enter>.
La función lee caracteres del teclado hasta que se encuentra el caracter de retorno de carro (pulsación de la tecla <enter>). Este caracter lo deja pendiente de proceso en el buffer de entrada. Al mismo tiempo destacar que scanf() devuelve el número de datos leidos. Veamos:
 

#include <stdio.h>

main()
{
   int datos_leidos, dia, mes, anyo;

   printf("Escribe una fecha separada por guiones: ");
   datos_leidos = scanf ("%d-%d-%d", &dia, &mes, &anyo);
   printf("scanf() ha leido %d datos\n", datos_leidos);
   printf ("La fecha es %d-%d-%d", dia, mes, anyo);
}


La función scanf() asigna un tipo de dato especificado entre comillas ("%<tipodedato>") a una direción de una variable especificada. Al mismo tiempo devuelve el número de datos leidos. Si no lee ningún dato devuelve un 0. Observa la tabla:
 

d

Puntero a int. Enteros  en base 10

o

Puntero a int. Enteros en base 8

x; X

Puntero a int. Enteros en base 16

i

Puntero a int. enteros en base 10, 16 u 8. si el entero comienza por 0, lo toma como en base 8. si cmienza por 0x ó 0X se toma en base 16.

u

Puntero a unsigned int. Enteros positivos (sin signo) en base 10.

f

Puntero a float. Valor con parte fraccionaria (float)

e, E

Puntero a float. Valor con parte fraccionaria en notación exponencial

g, G

Puntero a float. Valor con parte fraccionaria, toma el valor más corto, bien exponencial, bien sin exponente

c

Puntero a char. Leerá un sólo caracter

s

Puntero a char. Leerá una cadena de caracteres

n

Puntero a int. Lee el número de caracteres leidos del buffer o de un fichero especificado. Esto sería de la siguiente manera:
 

int num, num_car_leidos;
scanf("%d%n", &num, &num_car_leidos);
printf("He leido %d caracteres del buffer\n",  num_car_leidos);


Si intentamos asignar datos que no corresponden el tipo declarado en la variable con el tipo de dato especificado en la función scanf(), no los leerá y por tanto no habrá asignación. Un curioso caso es que cuando declaremos una variable int y queramos leer esa variable por teclado usando esta función, puede que nos equivoquemos y en vez de pulsar una variable de un número en nuestro teclado, pulsemos una letra o símbolo no numérico. En este caso scanf() lo interpretará como un número intentará asignarlo a la dirección de la variable especipicada. Para ello usaremos una función que nos libre de demejantes errores, pero eso será más adelante.
