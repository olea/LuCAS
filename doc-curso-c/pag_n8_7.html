8.7.- Paso de punteros a estructuras a funciones



En el programa anterior vimos como acceder a los elementos de una arrya mediante un puntero. Lo ideal ha de ser enviar el puntero a una función y que ella se encrgue de hacer con él lo que deba. Al poder enviar el puntero a una función, conseguimos "entregar" una dirección de memoria (paso por referencia) que tan sólo ocupa 4 bytes lo cuál nos da muchas ventajas encuianto a memoria se refiere. 

Nos situamos en el programa anterior y desarrollaremos una función que averigue el númerop de ladrillos totales "manejados" por el array estr. Se trataría recorrer mediante un bucle for() las diferentes variables n_ladrillos contenidas en las diferentes estructuras de que consta el array y sumarlas todas. Este programilla lo desarrollaremos usando una fución que "capture" el puntero al primer elemento de la estructura y vaya recorriendo el array y sumando el número de ladrillos. La función habría de ser similar a lo que sigue:
 

int sumaladrillos(el_puntero)

struct miestructura *el_puntero;

             {

              int i, sumador=0;

              for(i=0; i<2; i++)

                   {

                    sumador += el_puntero->n_ladrillos;

                    el_puntero++;

                   }

return sumador;

}





Cuando la función sumaladrillos() reecibe un valor, queremos que sea un puntero a una estructura de tipo miestructura, por eso declaramos a continuación

struct miestructura *el_puntero;
 

A partir de aquí sólo hemos de porceder como siempre, recorriend el array como hemos aprendido y alamacenando el valor que nos interesa en la variable sumador. Como fin de la función, sólo hemos de retornar el valor alamacenado en sumador.

El truco de este programa es que para que la función sumaladrillos() pueda reconocer el puntero el_puntero como un puntero a una estructura de tipo miestructura la declaración de esta estructura ha de estar en el exterior de main() a fin de que tanto main() como sumaladrillos() la puedan "conocer". El programa completo lo vemos a continuación:

 


#include <stdio.h>

 
struct miestructura

          {

            int n_ladrillos;

            char t_construct[20];

            int coste;

          };


 
 

/*Declaración de funciones */

int suma_ladrillos(struct miestructura *punt);
 
 

main()

        { 

          int n_total;
 

          struct miestructura estr[2]=

                 {

                      /*Definimos estr[0]*/

                     {

                      1000,

                      "chabola y muro",

                      120000

                     },

                       /*Definomos estr[1] */

                     {

                      500,

                      "Torreón",

                      30000
 

                     } 

                 };


 

         struct miestructura *punt=NULL;

/*asignamos la dirección de memoria del primer elemento del array en el puntero*/

         punt = &estr[0];

         n_total = sumaladrillos(punt);

         printf("Se han usado %d ladrillos en todo el array\n", n_total);

          }  
 
 

int sumaladrillos(punt)

struct miestructura *punt;

       {

        int i, sumador=0;

        for(i=0; i<2; i++)

            {

               sumador = sumador + punt->n_ladrillos;

               punt++;

            }

        return sumador;

       }


Este programa tiene la siguiente salida:
 


Se han usado 1500 ladrillos en todo el array



Con lo cual hemos podido averiguar el número de ladrillos usados en todo el array de estructuras.
