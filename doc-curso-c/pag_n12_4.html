12.4.- Agregar elementos en una lista delante o detrás de otro apuntado por *indice


Antes ya vimos como añadir elementos al final de una lista o al principio. Para añadir elementos entre nodos, hemos de usar como referencia el puntero struct datos *indice, y una vez situados podremos insertar otro nodo delante del mismo o detrás. Una vez situado el indice en el lugar de la inserción, debemos de elegir si insertarlo delante o detrás de indice. A continuación un código que inserta un nuevo nodo por delante de indice. Para ello suponemos que *indice está situado en el punto de inserción y queremos insertar el dato 123 delante de 19:



int nuevo_numero= 123;

nuevo_nodo = (struct datos )malloc(sizeof(struct datos));
*nuevo_nodo = *indice; /*copia todos los datos de indice en nuevo_nodo */
indice->siguiente = nuevo_nodo;
indice->numero = nuevo_numero;

Ya sólo queda insertar definitivamente el elemento en la lista. Esta acción es un poco extraña pero funciona bastante bien, ya que indice sigue apuntando al mismo lugar, pero con el contenido del elemento que queremos insertar. Hemos dejado el elemento a insertar por delante de indice. Para insertalo detrás de indice hariamos como sigue:



int nuevo_numero= 123;

nuevo_nodo = (struct datos )malloc(sizeof(struct datos));
nuevo_nodo->siguiente = indice->siguiente;
indice->siguiente = nuevo_nodo;
