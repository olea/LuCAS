8.3.- Anidamiento de estructuras



Supongamos que queremos que nuestra definicón de estructura sea más profesional, de manera que en su interior exista otra estructura que contenga las notaciones del nombre vulgar, familia, orden y nombre cientifico al que pertenece al animal en cuestión. Para ello, en primer declararemos la estructura que queremos anidar. La llamaremos struct nombres. A continuación declararemos la estructura priincipal y en su interior definiremos una estructura del tipo struct nombres. Como punto final introduciremos datos en ámbas estructuras y visualizaremos los datos. Como novedad fíjate que para referirnos a un elemento de la estructura anidada hemos de usra dos operadores de pertenencia. Veamos como hacerlo:



#include <stdio.h>


 
 

#define SI 1

#define NO 0
 
 

void main()

     {

               /* definimos la estructura nombres */

       struct nombres

                {

                 char n_vulgar[20];

                 char familia[20];

                 char orden[20];

                 char n_cientifico[20];

                };


 

       struct animal

               {

               /* definimos una variable nom_aguila de la struct nombres */

                struct nombres nom_aguila;

                int n_patas;

                char *pelaje;

                int pico;

                int mamifero;

               };


 

       struct animal aguila=
                {

                    {
                        /* Anidamiento de nom_aguila */ 

                      "Aguila real",

                      "Accipítridos",

                      "Falconiformes",

                      "Aguila chrysaetes",

                    },

                     2,

                     "plumas", 

                     SI,

                     NO

               };


 
 

      strcpy(aguila.nombre, "aguila");

      aguila.n_patas = 2;

      strcpy(aguila.pelaje, "plumas");

      aguila.pico = SI;

      aguila.mamifero = NO;

      strcpy(aguila.familia, "rapaces"); 


      puts("DATOS DE AGUILA\n");

 
                   //primer dato de la estructura anidada

      printf("Nombre vulgar: %s\n", aguila.nom_aguila.n_vulgar);

      printf("nº patas: %d\n", aguila.n_patas);

      printf("Tipo de pelaje: %s\n", aguila.pelaje);

 
      if (aguila.pico == SI)

                printf("Pico: SI\n");

      else

                printf("Pico: NO\n");

      if (aguila.mamifero == SI)

                printf("Mamifero: SI\n");

      else

                printf("Mamífero: NO\n");

                 // Visualizamos los demás datos de la estructura anidada

     printf("Familia: %s\n", aguila.nom_aguila.familia);

     printf("Orden: %s\n", aguila.nom_aguila.orden);

     printf("Nom. cientifico: %s\n", aguila.nom_aguila.n_cientifico);


}


 
 

Como dato interesante observa que en la declaración de animal hemos declarado a *pelaje como un puntero a char. Cuando inicializamos la estructura aguila ya le aplicamos un valor, que es "plumas". Si intentáramos asignar el valor "plumas" fuera de la estructura, por ejemplo de la forma strcpy(aguila.pelaje, "plumas") nos encontraríamos que el programa compilaría pero no funcionaría debido a la falta de espacio para almacenar la cadena "plumas". El problema es que aguila.pelaje apunta a null y este programa ya tiene destinada exactamente la cantidad de memoria que va a usar, por tanto cuando asiganmos una cadena nueva al puntero a char, lo que obtenemos es que la cadena que introducimos para nuestro puntero se sobreescriba sobre otras partes de nuestro programa. Para poder asiganar memoria para esta cadena hemos de usar la funcion malloc(). La salida del programa será:




DATOS DE AGUILA


Nombre vulgar: Aguila real

nº patas: 2

Tipo de pelaje: plumas

Pico: SI

Mamífero: NO

Familia: Accipítridos

Orden: Falconiformes

Nom. cientifico: Aguila chrysaetes
