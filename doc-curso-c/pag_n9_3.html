9.3.- Funcion pow() y sqrt()                                     



Para poder elevar un número a una potencia determinada se usa esta función.

#include <math.h>

double pow(double base, double exponente);

Devuelve el resultado de elevar double base a double exponente



La forma de usar esta función sería:



double base = 9;

double exponente = 2;

printf("resultado = %g\n", pow(base, exponente);



Que arrojará como resultado:

resultado = 81







sqrt()

Para averiguar la raiz cuadrada de un valor double determinado usamos la función sqrt()

#include <math.h>

double sqrt(double x);

Devuelve el valor double de la raiz cuadrada de x


En el caso de que el valor de x sea negativo, la función devuelve un error.



double x = 9;

printf("resultado = %g\n", sqrt(x);



Cuyo resultado es:


resultado = 3
