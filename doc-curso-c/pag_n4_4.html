4.4.- getchar(). Leer un caracter del teclado.



La forma de la función getchar() es la siguiente:
 

#include <stdio.h>

int = getchar();


Compatible con UNIX, MSDOS

De esta manera conseguimos asignar un caracter escrito por el teclado a la variable de tipo char.
Otra función que hace la misma operación es getchar(). Captura un caracter del teclado y asigna el resultado a una variable de tipo char.

Ya hemos visto que podemos leer un caracter del teclado mediante la función scanf() de la siguiente manera:
 

char car;

scanf ("%c", &car);


getchar() actúa de forma similar, pero es más fácil de escribir. Toma un caracter del buffer de entrada y lo asigna  y tal y como lo ha capturado  a una variable de tipo char.
 


/********* chareador.c ***************/

#include <stdio.h>

main()
   {
    char car;

    printf("\nEste programa traduce caracteres a código ASCII\n\n"");
    printf("Escribe un caracter en el teclado: ");
    car = getchar();
    printf("El caracter %c tiene el valor ASCII %d\n", car, car);
   } 



Este programa traduce caracteres a código ASCII

Escribe un carácter en el teclado: a

El carácter a tiene un valor ASCII 97



Cuando ejecutamos un programa queda abierto una especie de corriente de datos desde el teclado. Si tratáramos de leer un carácter desde el teclado y pulsáramos más de una tecla por ejemplo la cadena 'z13o0' y a continuación pulsamos <enter>, la función getchar() capturará el carácter z. La cadena restante '13o0' va a quedar pendiente de proceso. De tal manera que si ejecutemos de nuevo la función getchar(), y pusamos el carácter 'R' la función leerá el carácter z, ya que R se ha puesto a la cola de datos esperando ser procesada. Además también quedará almacenado el carácter nueva linea '\n', ya que es el carácter que se transmite al pulsar enter.
