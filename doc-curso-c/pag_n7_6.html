7.6.- Ordenación de cadenas de caracteres. Método de la burbuja


Con todas las funciones manipuladoras de cadenas de caracteres que conocemos, vamos a desarrollar un programa que ordene cadenas de caracteres por orden alfabético. Para ello usaremos una función main() muy cortita y las operaciones de capturar cadenas y ordenarlas se las dejaremos a funciones creadas para este fin. Puede ser un tanto compleja el la ordenación de cadenas, pero con sólo comprender el sistema de ordenado es suficiente. En el programa usaremos 4 funciones como puedes ver en la siguiente tabla:

int catch_cads(char *cad);

void ver_cads(char *cads[], int n_cads);

void orden_cads(char *punt_cads[], int n_cads);

void main()


Una función catch_cads() que se encarga de capturar cadenas por el teclado. Esta función recibe como argumento un puntero a char llamado cad. Lo que haremos en la función principal será asignar memoria para una cadena de caracteres y pasar la dirección de dicha cadena a la función catch_cad(), con el fin de que la rellene de carcteres. Devuelve un entero porque estamos interesados en que el entero que devuelva sea la longitud de la cadena, ya que si la longitud de la cadena es 0, tomarmos medidas. Estas medidas las veremos más adelante.

Usamos otra función ver_cads() que simplemente nos muestra las cadenas que tenemos almacenadas en el array cads[]. Las muestra por orden. Esta función sirve para comprobar que el programa funciona, pues visualizará las cadenas antes y despues de ordenarlas. Recibe como argumentos la dirección del array de cadenas y el número de cadenas que contiene el array.

Usamos una tercera función orden_cads(), que al igual que la anterior recibe como argumentos la dirección del array de cadenas y el número de cadenas. Esta función cambiará las cadenas de caracteres necesarias para que el array esté ordenado. Veremos más adelante el funcionamient de esto.

La cuarta e inevitable función, será main(), donde yace el programa y donde llamaremos a las anteriores funciones.

Empezaremos por desarrollar la función main(), para ver un poco más claro las esperanzas puestas en las funciones auxiliares, de main(): catch_cad(), ver_cads() y orden_cads(). En primer lugar declaramos un array de punteros a cadenas de caacteres char *cad[MAXIMO], array que ordenaremos más tarde. Un puntero a cad en cuya dirección almacenarermos la cadena a ordenar. Una variable int i que nos moverá por los vericuetos de los arrays y otra variable long_cad para recibir el tamaño de las cadenas. Como el número máximo de cadenas que capturaremos queda definido en #define MAXIMO. Esta será la función main():


#define MAXIMO 5


void main()

{

  char *cads[MAXIMO];

  char *cad;

  int i = 0;

  int long_cad;


  while(i < MAXIMO)

    {

     if(i==(MAXIMO-1))

        printf("Esta es la última cadena que te cojo. Ya no me caben más\n");

     if((cad = (char *)malloc(MAXFRASE)) == NULL)

        {

         printf("no memory for cad");

         exit(-1);

        }


     if((long_cad = catch_cads(cad)) != 0)//

        {

/* Hacemos espacio para una cadena */

         if((cads[i] = malloc(long_cad + 1)) == NULL)

            {

             printf("No memory");

             exit(-1);

            }

         strcpy (cads[i], cad);

        }

     else

        {

          printf("Has finalizado la introducción de cadenas\n");

          break;

        }

     i++;

    }//fin while()

    printf("Las cadenas capturadas son:\n");

    ver_cads(cads, i);

    printf("ordenando cadenas\n");
    system("clear");

    orden_cads(cads, i);

    puts("Estas son las cadenas ordenadas");

    ver_cads(cads, i);

    printf("\n\nastalavista\n");

}



El programa dispone de un bucle while que hace muchas operaciones importantes para nuestro programa. Lo inicializamos como while (i < MAXIMO) para que capture y almacene tantas cadenas como el valor de MAXIMO. Cuando vayamos a leer la última cadena porque ya no caben más en nuestro array le pasaremos un mensaje de aviso al usuario de nuestro programa. Esto lo conseguimos mediante la linea if(i==(MAXIMO - 1)).
Para asignar memoria a la dirección en la que almacenaremos nuestra primera cadena y siguientes usamos la línea if((cad = (char *)malloc(MAXFRASE)) == NULL), que en primer lugar asigna memoria equivalente al tamaño de MAXFRASE (tamaño máximo para cadena de caracteres), y especifica que devuelve un puntero a tipo (char *). Es aconsejable desde todos los puntos de vista especificar el tipo de valor al que apunta el puntero evuelto por malloc(). La función malloc() termina por devuelver la dirección de memoria donde aloja el tamaño de memoria solicitada. Si no pudiera reservar memoria para ello devolverá un puntero a NULL, con lo que mostraremos un mensaje y saldremos del programa.

Cuando capturemos una cadena puede ocurrir que no contenga caracteres (caso de que el usuario pulse <enter> antes de introducir algún carácter), en cuyo caso estimamos que el usuario no kiere introducir mas cadenas. La longitus de una cadena será escaneada por la función catch_cads(). Si el usuario pulsa algún carácter antes de pulsar <enter>, se ejecutará la sentencia if((long_cad = catch_cads(cad)) != 0)A continuación asiganará memoria para el primer puntero del array y por fín copiará la cadena cad en cads[i] mediante la función strcpy (cads[i], cad). Observa que si la longitusd de la cadena es 0 se ejecuta la sentencia else que nos expulsa del bucle while() con un mensaje de indicación. Como últimos dato de while() observar que su última línea suma una unidad a i para asignar ora cadena al array.

Continuando con la función main(). Observa los argumentos que le pasamos a ver_cads(cads, i). El nombre de la función nos indica que tan sólo se trata de visualizar cadenas. Realmente esta función no hace más que eso. Recibe como parámetros la dirección del array de cadenas y el número de elementos rellenados de dicho array. A continuación ejecutamos al función orden_cads(), con identicos parámetros que orden_cads(). Despues de todo reciben como argumentos una dirección de memoria, las funciones saben dónde han de dirigirse.

La primera función que hemos usado es int catch_cad(char *cad). Veámosla:

int catch_cads(char *cadena)

{


  printf("Escribe una palabra (para terminar pulsa <enter>: ");

  gets(cadena);

     return 0;

  return strlen(cadena);

Esta función recibe como argumento la dirección de una cadena de caracteres para rellenarla. A partir de aquí. Sólo hay que usar la función gets(cadena), función muy socorrida para rellenar cadenas de caracteres. Si el usuario del programa pulsa la tecla <enter> antes de introducir algún carácter, estimamos que no quiere introducir más cadenas. En esta caso cadena apuntaría a NULL y devolveremos el valor 0, caso contrario devolveremos la longitud de la cadena de caracteres. Observa lo que pasa en la función main() cuando la función catch_cad() devuelve 0: Da un mensaje de confirmación y sale del bucle while mediante break. Otro dato curioso es que si la longitud de la cadena es mayor que 0, la función devuelve el valor devuelto por otra función: strlen(cadena). Ocurre que con el valor devuelto por esta función podremos reservar la memoria necesaria con la función malloc(). En la función main() reservamos un byte más de memoria a fin de que nos quepa el carácter nulo de fin de cadena.

La siguiente función es ver_cads():

void ver_cads(char *cads[], int n_cads)

{

Recibe como argumentos un array de punteros a cadenas de caracteres char *cads[] y el número de elementos introducidos en el array de punteros a cadenas n_cads. Para visualizar las cadenas del array, sólo hemos de recorrernos el array usando un bucle for() como en el ejemplo. Hemos usado la notación de punteros para visualizar las cadenas de caracteres, pero podriamos haber usado la fórmula puts(cads[indice]); que como hemos visto anteriormente origina resultados semejantes e igual de efectivos.



El algoritmo de ordenamiento de la burbuja

Existen muchas maneras de ordenar arrays y datos, incluso el lector puede inventar una y aplicarla a sus propìos programas. Uno de tantos métodos usados es el procedimeinento de la burbuja. Este algoritmo recorre tantas veces el array como elementos dispone dicho array y cada vez que se encuantra un elemento superior en valor al comparado lo va situando a continuación del inferior, de tal manera que los elementos mayores van situandose en un extremo del array (burbujeando), mientras que los inferiores van quedando "abajo". En todo caso este algoritmo sirve para colocar en orden ascendente y decendente. Nosotros colocaremos nuestras cadenas en orden de menor a mayor. Trata de comprenderlo y si eres capaz de asimilarlo completamente mejor que mejor, en todo caso el objetivo es que sepas como se puede ordenar un array y que cuando necesites ordenar alguno sepas donde hay un algoritmo para ello. No te preocupes si no puedes comprender de un vistazo el método de ordenación. Para autoayudarte puedes escribir en un papel el nombre de las variables y hacer las iteraciones necesarias a lapicero, y verás como realmente las ordena. La función es:


void orden_cads(char *punt_cads[], int n_cads)

{

   char *aux;

   int i;


   n_cads++;

   while(--n_cads)

      {

       for(i=1; i < n_cads; i++)

       if(strcmp(punt_cads[i-1], punt_cads[i]) > 0)

           {

            aux = punt_cads[i-1];

            punt_cads[i-1] = punt_cads[i];

            punt_cads[i] = aux;

           }

      }

}





Hemos sumado una unidad más al valor n_cads recibido en orden_cads() a fin de que el bucle while() se recorra todas las las cadenas hasta que n_ cads adquiera el valor 0 (false). Cuando n_cads es recibido como argumento por la función orden_cads(), entra valiendo 4 y nosotros necesitamos que recorra el array 5 veces. Este es el motivo de que sumemos una unidad a esta variable. Intenta ejecutar el programa comentando la línea n_cads++; y observarás que el último elemento que introduces no es ordenado en ningún caso. Podríamos lograr el mismo objetivo sin sumar una unidad a n_cads, poniendo una condición diferente en el bucle while de la forma:

while (n_cads > = 0)

Fíjate tambien que hemos declarado un puntero a char llamado aux y que hemos escogido precisamente un puntero a char porque el contenido del array de punteros es precisdamente eso, un array de direcciones de memoria tambien llamado un array de punteros a char y esta variable auxiliar nos va a servir para almacenar temporalmente valores punteros a fin de ordenarlos a nuestra conveniencia.

Este array ordena en orden ascendente, si quisieramos ordenar en orden desdendente, observa que sólo hemos de cambiar el valor comparativo de la sentencia if() por

if(strcmp (punt_cads[i-1], punt_cads[i]) < 0)

De esta manera cada vez que detecte que la comparación entre dos elementos del array es menor que 0, colocará la cadena de menor valor al final de del array, la "burbujeará" hacia el final.

La sentencia while(--n_cads) va estar funcionando hasta que el valor de n_cads sea 0 (false).
