5.1.- El almacenamiento extern y una pequeña función

Los tipo de almacenamiento de datos en una variable viene especificado por los identificadores:

  • extern

  • auto

  • static

  • register


Veamos a continuación un programa que nos ilustrará este capítulo:

Este programa tiene la siguiente salida:

Dime el número atómico de Arsénico

Escibelo a continuación: 36

Escribe otro nº atómico por favor: 33
Muchas gracias, ya no estoy confuso, astalavista\n



Observa como las funciones main() y analiza() conocen el valor de n_atomico. En la función main() no la hemos declarado, y hacemos referencia a ella sin ningún tipo de problema. Cuando hacemos mención a n_atomico todo funciona perfectamente. Al anteponer la palabra clave extern en el interior de la función analiza(), linea 17, lo que hacemos es redundar en la declaración. Tal vez podamos pensar que es hacer más código de la cuenta, pero nos servirá para que cuando "leamos" el programa tengamos los conceptos más claros y sepamos que n_atomico es una variable extern.

Si tuviéramos otro fichero donde quisiéramos acceder a la variable extern, podríamos declararla en el exterior de toda función en cuyo caso no sería necesario anteponer el identificador extern o en el interior de alguna función, en cuyo caso debemos de especificar el identificador extern. Las variables extern si no son inicializadas, se inicializan automáticamente a 0. Imaginemos que el programa anterior le tenemos repartido en dos ficheros. Uno de ellos contendrá la función main() y lo vamos a llamar main.c. El segundo fichero contendrá la función analiza() y lo llamaremos analiza.c. Veamos como declararíamos la variable n_atomico  y el aspecto de los dos ficheros:
 
 

/*****main.c*****/

#include<stdio.h>

int n_atomico; /* Esta variable no está inicializada. C la inizializa a 0 */ 
void analiza(void);

main()
      {
/* Esta función conoce el valor y la dirección de la variable extern n_atomico   */

       printf("Estoy un poco confuso\n");
       printf("Dime el nº atómico del Arsénico?\nEscríbelo a continuación: ");
       scanf("%d", &n_atomico);
       while(n_atomico != 33)
             analiza();
       printf("Muchas gracias, ya no estoy confuso, astalavistabeibi");
      }
 


 


/*****Este es el fichero analiza.c ***********/

#include <stdio.h>

extern int n_atómico;

void analiza()
    {
     extern int n_atomico; /*Esta declaración es redundante, se podría omitir. Si hacemos esta declaracion podemos omitir la exterior */
     printf("mmmmmm..., creo que ese no es.\n Escribe otro nº atómico por favor: ");
     scanf("%d", &n_atomico);
    }


Para la compilación de estos dos ficheros procederíamos de la siguiente manera:
 

    ~/ gcc  main.c analiza.c -o arsenico.bin


Esta orden generaría un fichero ejecutable llamado arsenico.bin.
