9.6.- Funciones de tiempo                                      



clock()



Sinopsis de clock()

#include <time.h>

clock_t clock (void);



clock_t está declarado typedef long clock_t.

Esta función devuelve el número de pulsaciones de reloj del procesador en la ejecución del programa que llama a esta función. La constante CLOCKS_PER_SEC es la encargada de almacenar el número de pulsaciones por segundo del procesador. Entonces si quisieramos obtener el número de segundos transcurridos desde la ejecución del programa, dividiríamos el valor devuelto por clock() entre la constante CLOCKS_PER_SEC:

int segundos = clock() / CLOCKS_PER_SEC

Y como muestra de como funciona clock() y la constante CLOCKS_PER_SEC puedes echar un vistazo al siguiente programa.

Función time()

La función time() devuelve el número de segundos transcurridos desde las 0 horas del 1 de enero de 1970.



Sinopsis de time()

#include <time.h>

time_t time (time_t *segundos);



time_t está definido como typedef long time_t.



Función ctime()

Esta función convierte un valor time_t (long) en una cadena de caracteres comprensible. La cadena de caracteres tiene la forma:

[dia de la semana] [mes] [dia del mes] [hora] [año]

Sinopsis de ctime()

#include <time.h>
char *ctime(time_t *segundos);

Devuelve un puntero a la cadena de caracteres que resulta. time_t es un valor long



Un pequeño ejemplo del uso de esta función podría ser:

#include <stdio.h>

#include <time.h>


main()

       {


       long segundos;

       /* Asignamos en &segundos el tiempo transcurrido desde el 1-1-1970*/

       time(&segundos);

       /*Pasamos a ctime() el valor almacenado pot time() en &segundos */

       printf("Fecha y hora = %s\n", ctime(&segundos));

}



En el momento de ejEcución de este programa en nuestra máquina, el resultado ha sido:

Fecha y hora = Mon Dec 13 13:55:21 1999






Función localtime()

Esta curiosa función despieza el número de segundos pasados en su único argumento, en variables de tipo int previamente definidas en la estructura tm en el fichero time.h. La estructura tm tiene el siguiente aspecto en nuestro sistema:

struct tm


{

  int tm_sec;               /* Segundos. [0-60] (1 leap second) */

  int tm_min;               /* Minutos. [0-59] */

  int tm_hour;              /* Horas. [0-23] */

  int tm_mday;              /* Día. [1-31] */

  int tm_mon;              /* Mes. [0-11] */

  int tm_year;              /* Año - 1900. */

  int tm_wday;              /* Día de la semana.[0-6] */

  int tm_yday;              /* Día del año.[0-365] */

  int tm_isdst;             /* DST. [-1/0/1]*/


# ifdef __USE_BSD

  long int tm_gmtoff; /* Seconds east of UTC. */

  __const char *tm_zone; /* Timezone abbreviation. */

# else

  long int __tm_gmtoff; /* Seconds east of UTC. */

  __const char *__tm_zone; /* Timezone abbreviation. */

# endif

};



Los enunciados de #ifdef , #else no hace falta que los comprendas, más adelante veremos como se usan, Tan sólo se trata que observemos como está definida struct tm.



Sinopsis de localtime()

#include <time.h>

struct tm *localtime(long *segundos);

Esta función devuelve un puntero a la estructura tm, Dando valores a todas Sus variables internas. Si no puede hacer la conversión devuelve un puntero nulo.



Para usar esta función es necesario declarar un puntero a una estructura tm y luego capturar el valor que nos interese. A continuación vemos un ejemplo en el que capturamos todos los valores de struct tm.

#include <stdio.h>

#include <time.h>


main()

{



       struct tm *tiempo; /*Declaramos un puntero a una estructura de tipo tm*/

       long segundos;

       time(&segundos); /* Adjudicamos a segundos el tiempo actual*/

                   tiempo = localtime(&segundos); /* Damos valores a la estructura apuntada por tiempo*/



       printf("Fecha y hora = %s\n", ctime(&segundos));

       printf("Segundos = %ld\n", tiempo->tm_sec);

       printf("Minutos = %ld\n", tiempo->tm_min);

       printf("Horas = %ld\n", tiempo->tm_hour);

       printf("Dia = %ld\n", tiempo->tm_mday);

       printf("Mes = %ld\n", tiempo->tm_mon);

       printf("Año = %ld\n", tiempo->tm_year);

       printf("Dia Sem = %ld\n", tiempo->tm_wday);

       printf("Dia año = %ld\n", tiempo->tm_yday);


}



La salida del programa es:

Fecha y hora = Mon Dec 13 14:36:15 1999


Segundos = 15

Minutos = 36

Horas = 14

Dia = 13

Mes = 11

Año = 99

Dia Sem = 1

Dia año = 346



Como podemos ver este programa de ha ejecutado un 13 de diciembre, para nosotros mes nº 12, sin embargo la lectura de tiempo->tm_mon es el mes 11. La razón es que tm_mon comienza a contar los meses desde 0. El mes 0 corresponde a enero y el mes 11 a diciembre.
