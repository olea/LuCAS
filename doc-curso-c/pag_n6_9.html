6.9.- Bucle for()                                           



Todavía no hemos visto ningún ejemplo de esta útil sentencia. Pero a partir de este punto aparecerá con cierta frecuencia, por lo que es fundamnetal que le prestes la atención que se merece. su sintaxis es:
  

for(<inicialización>; <evaluación de la condición>; <incremento o decremento>)
 {
 // sentencias
 }


Observemos un ejemplo sencillo del uso de este interesante bucle:
 

/****************michacha.c******************/

#include <stdio.h>

main()
      {
       int i;
       for (i=0; i<5; i++)
            {
             printf("%d.-   Mi chacha tiene marcha\n", i);
            }
      }


La salida del programa es:
 
 

0.-   Mi chacha tiene marcha
1.-   Mi chacha tiene marcha
2.-   Mi chacha tiene marcha
3.-   Mi chacha tiene marcha
4.-   Mi chacha tiene marcha


Este bucle for se ejecuta en 4 etapas:
La primera es una inicilización. En nuestro caso inicializa la variable int i a 0 mediante la sentencia de asignación i = 0; que está contenida en el interior de los paréntesis de for(). En este momento el valor de i es 0.
La  segunda etapa es una condición necesaria para que se ejcuten las sentencias contenidas entre las llaves de for(). Esto significa  que cada vez que esta condición sea verdadera, se imprimirá por la pantalla la frase relativa a mi chacha.
Puesto que la condición es cierta no queda más remedio que ejecutar la tercera etapa de nuestro bucle, que es ejecutar las  sentencias que contienen el bucle for() y que estan encerradas entre llaves. Observa que en la función printf() tambien aparece reflejado el valor de i. La primera línea del resultado de la ejecución de  michacha es
 

0.-   Mi chacha tiene marcha


Observa que el número 0 es el valor de i, lo que significa que la variable int i aún no se ha incrementado. En su tercera etapa for() incrementará en una unidad el valor de int i y de nuevo comparará si int i es menor que 5 y si esa condición es cierta de nuevo ejecutará el interior de su bucle. Otra variante de nuestro programa michacha.c podría ser:
 

/****************michacha.c*************************/

#include <stdio.h>

main()
      {
       int i;
       for (i = 5; i > 0; i--)
            {
             printf("%d.- Mi chacha tiene marcha\n", i);
            }
      }


cuya salida sería: 
 
 

5.-   Mi chacha tiene marcha
4.-   Mi chacha tiene marcha
3.-   Mi chacha tiene marcha
2.-   Mi chacha tiene marcha
1.-   Mi chacha tiene marcha


Date cuenta que aquí no ocurre como con do while() donde al menos una vez se ejecuta el bucle a pesar de que la condición no sea cierta, obsérvalo. hemos desarrolado dos programillas sencillos, uno cuanta hacia arriba y el otro hace una cuenta hacia atrás. El bucle for() es muy versátil y admite una gran variedad de argumentos, incluso funciones. Vamos a ver algunos programillas que nos den más luz sobre for().

Los matemáticos son gente rara y extraña, que han desarrollado enormes y laboriosas teorías que nos hacen la vida más fácil a los demás mortales. Uno de ellos se llamaba Leonardo Fibonacci (1170 - 1240). Era un matemático italiano que desarrolló muchas teorías sobre series de números. Era muy admirado por sus compatriotas, entre otras cosas porque exponía sus duros, ingeniosos y originales problemas matemáticos en forma de relatos, de manera que la mayoría de sus oyenten entendían los plantamientos. Parece ser que incluso el ayuntamiento de la ciudad de Pisa le concedió un salario anual por sus servicios prestados. Entre otras cosas, desarrolló y descubrió la hoy llamada serie de números de Fibonacci, en la cual cada número de la serie es igual a la suma de los otros dos anteriores. Este sería el programa para desarrollar la serie de Fibonacci:
 
 

/*************** fibonacci.c***************/

#include <stdio.h> 

main()
  {
  int i, resp, num, prenum;
  for (i=1, num = 0, prenum = 1; i < 21; i++) 
     {
      resp = num + prenum; 
      printf("posicion %d ---> %d\n", i, resp); 
      prenum = num; 
      num =resp; 
      }
  printf("\n\nastalavista\n"); 
 }

Se puede observar en este programa como en el bucle for() son inicializadas tres variables a la vez: i, num y prenum. i nos servirá para establecer un número de orden.Almacenaremos los valores num y prenum en la variable resp, y el valor de esta variable será el que se mostrará por pantalla por cada unidad de iteración de for(). A continuación asignamos el valor de num a prenum. Y el valor de resp es asignado a num, de manera que ya queda listo el bucle para que cuando se ejecute la sentencia:
 
 

resp = num + prenum; 



quede asignado en resp la suma de los dos últimos números mostrados por pantalla.

Tambien existen bucles for() anidados en otro for().
 

#include<stdio.h>

  main( )
  {
  int i , j, respuesta;
  char opcion = 's'; 
  for(i = 1; i <= 10 && opcion != 'n'; i++) 
         {
         for(j = 1; j <= 10; j++) 
               { 
               printf("\nEscribe el resultado de %d x %d = ", i, j); 
               scanf("%d", &respuesta); 
               if (respuesta != i*j) 
               printf("\n¡¡OHHH! error\n"); 
               else printf("\n¡¡¡¡correcto!!!!!\n"); 
               }
        printf("\n Quieres otra tabla de multiplicar? "); 
        fflush (stdin); 
        opcion = getchar(); 
        printf("\n"); 
        } 
  printf("Gracias por usar nuestro servicio matemático\n"); 
  } 

 



El anterior programa muestra por pantalla la tabla de multiplicar, comenzando por la del 1 y terminado con la tabla el 10. Esto lo consigue con el primer bucle for(). Para que este bucle se pueda ejecutar han de cumplirse dos condiciones: que int i sea menor o igual que 10 y que el valor de respuesta sea diferente de 'n'. De esta forma no nos mostrará tablas de mltiplicar superiores a la del 10 y además el usuario estará de acuerdo en mostrar la tabla. Hemos inicializado la variable respuesta con el valor 's', a fin de que no comience con un valor basura y se de la extrña circunstancia de que el valor basura sea 'n', en cuyo caso no se ejecutaría el bucle.
Ya estamos en el bucle for() anidado, Observa que todo marcha estupendamente, pregunta por la respuesta de la multiplicación correspondiente y nos responde si la respuesta es acertada o erronea. Cuando int j alcance el valor de 10, el bucle for() anidado terminará y pasará a ejecutarse la siguiente sentencia del bucle principal, en la que nos pregunta si deseamos continuar con la siguiente tabla de multiplicar. En el caso de que respondamos n cuando lleguemos al bucle principal, sencillamente no se ejecutará.

Otra de las variantes de for() es usar funciones. Observa que para que se ejecute el bucle, de lo que se trata es de que la condición devuelva un 0. Inventaremos un programa en la que para finalizar, exista una función que devuelva 0 a la condición de for().

Echemos un vistazo al bucle for() de la función main(). La "inicialización" es símplemente la presentación del programa por medio de la llamada a la función torpedo(). A continuación ha de ejecutarse una condición. Hemos de tener en cuenta que para que se pueda ejecutar el bucle for() el valor devuelto por la condición ha de ser verdadero (distinto de 0). Para poder parar el programa no se necesita otra cosa que la función submarino() devuelva como valor a misil el valor 0. Cuando se ejecute la función for() simplemente nos muestra por pantalla la siguiente cadena de caracteres:
