10.9.- Escritura y lectura de datos en bloques, fwrite() y fread()


Es muy interesante escribir estructuras de datos en ficheros en modo binario, pues a diferencia de fprintf(), fwrite() gasta la mínima cantidad de memoria necesaria. La escritura de datos en estructuras, nos permite acceder a los datos alamcenados en el fichero de forma un tanto ordenada, veamos la sinopsis de estas funciones y el método de uso:

Sinopsis fwrite() y fread()

#include <stdio.h>
size_t fwrite(const void *ptr, size_t longitud, size_t n_elem, FILE*stream);

Devuelve el número de elemntos escritos. Si el número es menor que n_elem es que ha habido algún error

size_t fread( void *ptr, size_t longitud, size_t n_elem, FILE *stream);

Devuelve el número de elemntos leidos. Si el número es menor que n_elem es que ha habido algún error


size_t longitud es el número de bytes que deseamos escribir en el fichero apuntado por stream procedentes de const void *ptr. Análogamente ocurre lo mismo en la función fread().


La forma de escribir dos variables de diferente tipo en un fichero sería:



FILE *stream;
char car;
cadena[10];
float numero;
.....
.....

fwrite(&car, sizeof(car), 1, stream);
fwrite(cadena, sizeof(cadena), 1, stream);
fwrite(&numero, sizeof(numero), 1, stream);


Este código entresacado, si se ejecutara, escribirá en modo binario y consecutivo la variable car, la variable cadena y la varible float numero. Si quisiéramos leer los datos almacenados en el fichero por el programa podría ser algo paracido a esto:

FILE *stream;
char car;
cadena[10];
float numero;
.....
.....
/* suponemos que el fichero ha sido bierto en modo lectura y escritura /
fwrite(&car, sizeof(car), 1, stream);
fwrite(cadena, sizeof(cadena), 1, stream);
fwrite(&numero, sizeof(numero), 1, stream);

.....
.....
.....

/* Situamos el puntero de L/E en la posición 0 */

rewind();

fread(&car, sizeof(char), 1, stream);
fread(cadena, sizeof(cadena), 1, stream);
fread(&numero, sizeof(numero), 1, stream);


Este código no hace gran cosa, pero si podemos incrustar estas lineas en un programa y las varibles char car, char cadena[10], y float numero son cambiadas de valor antes de la ejecucion de cualquier función fread(), ocurrirá que al ejcutar fread() adquiriran de nuevo los valores asignados en un principio, pues estos son los datos almacenados en el fichero.
