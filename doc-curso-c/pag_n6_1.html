6.1.- Introducción



Durante el desarrollo de un programa es probable que encontremos que necesitamos repetir ciertas operaciones un determinado número de veces o hasta que se cumpla una condición dada. Un programa tomará ciertas decisiones en función de ciertos resultados. Diseñaremos el programa de tal forma que en función de ciertas respuestas del usuario ciertos valores de determinadas variables el programa tome una decisión, ejecute una serie de sentencias, repita una serie de sentencias un marcado número de veces, etc. Para ello C usa los bucles y las sentencias. Aunque ya hemos visto algún bucle como while() y ciertas sentencias como  if(), en este capítulo vamos a explicar todas las maneras de gobernar un programa. Los bucles y las sentencias de las que disponemos en C son:

  • Sentencia  if() y else

  • Sentencia switch()

  • Sentencia break

  • Sentencia exit(0)

  • Sentencia continue

  • Sentencia system

  • Bucle while() y sentencia do

  • Bucle for()
