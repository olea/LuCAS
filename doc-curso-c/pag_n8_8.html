8.8.- Uniones                                              



Una unión es una variable que puede albergar a datos de diferentes tipos, pero no contiene más de un sólo dato, el dato que nosotros queremos que contenga. Este tipo de construcciones las usamos para tomar el dato que más nos interesa de un objeto determionado para procesrlo a nuestro gusto. La forma de referirnos, declarar y visualizar elementos de una unión es idética a las estructuras. Observemos en un ejemplo el funcionamiento de union:


 


/*********************union.c*********************/

#include <stdio.h>

main()

           {

                /* Esta es la union */

             union mes

                 {

                  char nombre[20];

                  int dias;

                 };

              union mes t_mes;

              union mes *punt;

              printf("sizeof(union mes) = %d\nsizeof(t_mes) = %d\n",     sizeof(struct mes), sizeof(t_mes));

              punt = &t_mes;
 
 
 
 

              printf("Escribe el nombre del mes: ");

              scanf("%s", punt->nombre);

              printf("Escribe los dias del mes %s: ", t_mes.nombre);

              scanf("%d", &punt->dias);
 
 

              printf("punt->dias = %d\n", punt->dias);

              printf("punt->nombre = %s\n", punt->nombre);

 }


 

A continuación la salida del programa:
 


sizeof(struct mes) = 20

sizeof(t_mes) = 20

Escribe el nombre del mes: septiembre

Escribe los dias del mes septiembre: 30


punt->dias = 30

punt->nombre =


Una de las características importantes de las uniones es que tienen el tamaño de elemento mayor. En el programa union.c la composición union mes contiene las siguientes variables:

-char nombre [20]; tiene un tamaño de 20 bytes
-int dias; tiene un tamaño de 4 bytes (2 bytes en Windows'xx')

Si union mes fuera un estructura tendría un tamaño de 24 bytes (22 en Windows'xx'), sin embargo union mes es una union y una de las características de las uniones es que tienen el tamaño del miembro mayor. En nuestro caso union mes mide 20 bytes que es el tamaño del miembro mayor char nombre[20]. Cuando introducimos datos en una unión sólo tendrá validez el último dato validado, ya que este último dato sobreescribe a los anteriores. En el programa que nos ocupa el último dato introducido es punt->dias, cuando intentamos visualizar punt->nombre, la lectura queda en blanco, pues punt->dias se ha sobreescrito sobre punt->nombre.
