9.5.- Funciones fabs() y abs()                                     



Estas funciones devuelven el valor absoluto de un entero o un float:

Sinopsis de abs()

#include <math.h>

unsigned int abs(int num);

Devuelve el valor absoluto de int num



Sinopsis de fabs()

#include <math.h>

double fabs(double num);

Devuelve el valor absoluto del numero decimal double num





Un ejemplo de uso de estas funciones es:



#include <stdio.h>

#include <math.h>


main()

{

  int i=-1;

  float j = -2.45;


  printf("El valor absoluto de %d es %d\n", i, abs(i));

  printf("El valor absoluto de %f es %f\n", j, fabs(j));

}



La salidadel programa es:



El valor absoluto de -1 es 1

El valor absoluto de -2.450000 es 2.450000
