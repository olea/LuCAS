9.4.- Funciones trigonométricas                                     


El valor de los argumentos de las funciones trigonométricas hemos de darlos en radianes. A continuación una enumeración de todas las funciones trigonométricas:

#include <math.h>

double sin (double ang);

Devuelve el seno de ang

double cos (double ang);

Devuelve el coseno de ang

double tan(double ang);

Devuelve la tangente de ang

Double acos(double cos);

Devuelve el arco entre 0 y p cuyo coseno sea el valor de double cos.

Double asin(double sin);

Devuelve el arco entre -p /2 y p /2 cuyo seno sea el valor de double sin.

Double atan(double tan);

Devuelve el arco entre -p /2 y p /2 cuya tangente valga double tan
