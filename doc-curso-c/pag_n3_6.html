El operador NOT, devuelve 0 si el operando sobre el que actúa es distinto de 0 y devuelve un 1 si el operando es 0. Observa este programa donde aparecen implementados los operadores logicos AND, OR y NOT :



#include <stdio.h>

main()

{
  int saca = 10, puntas = 0, lapiz;
  lapiz = saca && puntas;/*saca && puntas devuelve 0, asigna 0 a lapiz */

  printf("lapiz = %d\n", lapiz);


  lapiz = saca || puntas; /*saca || puntas devuelve 1, asigna 1 a lapiz */

  printf("lapiz = %d\n", lapiz);
lapiz = !(saca || puntas);/*saca || puntas devuelve 1,el operador '!' convierte el 1 a 0.Asigna 0 a lapiz */

  printf("lapiz = %d\n", lapiz);
}


El resultado de este programa es:

Operadores relacionales



Cuando queramos comparar valores para saber si uno es mayor que otro, igual, o simplemente distinto, usaremos los siguientes operadores:


==

Evalua si el primer valor es igual que el segundo en caso afirmativo devuelve 1. En caso negativo (false) devuelve 0. 

!=

Evalúa si el primer valor es distinto que el segundo. Devuelve 1 en caso afirmativo. Devuelve 0 en caso negativo

<

Evalúa si el primer valor es menor que el segundo. Devuelve 1 en caso afirmativo. Devuelve 0 en caso negativo

>

Evalúa si el primer valor es mayor que el segundo. Devuelve 1 en caso afirmativo. Devuelve 0 en caso negativo 

<=

Evalúa si el primer valor es menor o igual que el segundo. Devuelve 1 en caso afirmativo. Devuelve 0 en caso negativo

>=

Evalúa si el primer valor es mayor o igual que el segundo. Devuelve 1 en caso afirmativo. Devuelve 0 en caso negativo



Observa este programa: 



#include <stdio.h>

main()

{
  int exp = 1, resion = 3, res;
  res = exp > resion;/* exp > resion devuelve 0, asigna 0 a res */
  res = exp <= resion;/* exp <= resion devuelve 1, asigna 1 a la variable res */
  res = exp != resion/*exp != resion  devuelve 0, asigna 0 a la variable res */
}

En la primera sentencia el compilador hace lo siguiente: toma el valor de lewis, toma el valor de calimocho, suma ambos valores y el resultado lo asigna a la variable suciedad. Después sustituye %d por el valor de suciedad. Hemos cambiado el valor de una variable dentro de una función printf().

En la segunda sentencia suma el valor de lewis al valor de calimocho. Después sustituye %d por el valor de esta suma, pero no asigna nada a ninguna variable, ni tan siquiera a la variable porquería aunque existiera. si a pesar de esta explicación no estuviera lo suficientemente claro, no te preocupes, dentro de pocas horas, estarás listo y este pequeño contratiempo se te habrá olvidado. Operadores incremento (++) y decremento (--)

Para comprender estos operadores, vamos a intentar comprender una nuevo bucle, el bucle while(). Aunque más adelante profundizaremos más sobre ella, observa lo siguiente:




#include <stdio.h>

main ()
{
  int cont=0;

  printf ("Esta es la tabla del 5\n");
  while (cont <= 10)
     {
       printf ("5 * %d = %d\n", cont, 5*cont);
       cont = cont +1;
     }

}

La sentencia while() (while significa "mientras" en español) se ejecutará hasta que deje de cumplirse la condición que contenga el interior del paréntesis. En este caso, cuando la variable cont tenga un valor superior a 10 dejará de ejecutarse pasando a la suiguiente línea que coincide que en este caso no existe, por tanto se termina el programa. Viene a funcionar de la siguiente manera:

Primero comprueba que la equivalencia que hay en el interior de sus paréntesis es correcta. Si dicha equivalencia es verdadera (distinta de 0), pasará a ejecutarse la totalidad de las sentencias encerradas entre llaves. El bucle while(), dicho de otro modo ejecutará el contenido de las llaves hasta que el interior de los paréntesis sea falso o valor 0. Tengamos en cuenta que falso = =0. Un valor verdadero es en C cualquier valor distinto de 0.

Como el paciente lector se supondrá esta modificación, se debe al operador ++ usado en la linea 6. El hilo del programa analiza en primer lugar si la variable int cont es menor o igual que 10. Si esta evaluación es cierta, procederá a sumar una unidad a cont para a continuación ejecutar las sentencias contenidas entre las llaves del bucle while(). Por tanto, cuando cont valga 10, el programa lo evaluará y a continuación incrementará el valor de cont en 1 (cont = =11) y ejecutará las sentencias del bucle while() , mostrando a cont con valor de 11.

Sin embargo, si tuviéramos el operador ++ delante de cont (++cont), en primer lugar incrementaría en una unidad el valor de cont, luego lo compararía con 10, con lo que el programa volvería a la normalidad. En todo caso esta ultima modalidad no mostraría el valor 5 * 0.

    El operador --, funciona exactamente igual que el operador ++, con la particularidad que  en vez de sumar una unidad, decremento una unidad en la variable.

    ¡Importante! Hemos de tener un cuidado especial en el empleo de estos operadores de incremento.


int cont =2;
printf("%d, %d",  cont, cont*cont++);



Estas lineas de código, aparentemente deberían de mostrar por pantalla el resultado

2, 4



Sin embargo veremos por pantalla el siguiente resultado:

3, 4



Ocurre que el operador de incremento dispone de una gran preferencia, por tanto va a ejecutar en primer lugar el código cont * cont++, debido a que es donde está el operador de incremento. El siguiente paso será ejecutar cont * cont que arroja el resultado de 4 (cont vale 2 en estos momentos). Entonces se procede a sustituir el 2º %d por el valor 4. Una vez hecho esto procederá a sumar una unidad a cont y a sustituir el valor incrementado de cont por el primer %d. En todo caso no tenemos una garantía de que esto funcione así en todos los compiladores por lo que Como norma general, no operadores de incremento o decremento sobre una variable si esta está repetida en dicha línea.

Operador condicional (?:)



#include <stdio.h> 

main()

{
  int numero_1 = 3;
  int numero_2 = 1350;
  int menor;

  menor = (numero_1 > numero_2) ? numero_2 : numero_1;
  printf("el menor es %d\n", menor);
}


Este es uno de los operadores más raros y extraños de C, Es un if disimulado pero muy efectivo y fácil de usar. Además se escribe más fácil que un if().

Observa la expresión:



menor = (numero_1 > numero_2) ? numero_2 : numero_1;


El compilador analiza en primer lugar la expresión encerrada entre paréntesis

numero_1 > numero_2

Si esta expresión es cierta (TRUE), asignará a la variable menor el valor a continuación del símbolo '?', que en este caso coincide que es la variable numero_2 . Si la expresión evaluada fuera false (FALSE), asignará a la variable menor el contenido de la variable escrita a partir del símbolo ':'   que en este caso coincide con el valor de numero_1. En este caso el programa asigna a la variable menor el valor de numero_1.



Operador dirección de memoria (&). Contacto con punteros

Cuando definimos una variable sea del tipo que sea, el compilador reserva un espacio en la memoria del computador del tamaño indicado por el tipo de variable. El contenido de ese espacio de memoria será el valor de la variable definida. Evidentemente el programa sabrá en todo momento dónde está situada la variable, ya que tiene en memoria la dirección de la misma. Observa el siguiente código:


#include <stdio.h>

main()

{

  int num = 12;

  printf("El valor de num es %d\n", num);

  printf("La dirección de num es %p\n", &num);

}


Cuando visualicemos estas sentencias veremos algo parecido a esto:


El valor de num es 12

La dirección de num es 0xbffff7a4


El identificador %p es el que lee direcciónes de memoria. La dirección de memoria de la variable num en el caso de nuestro sistema es 0xbffff7a4 . Cuando lo ejecutes en tu sistema probablemente obtendrás una dirección de memoria distinta, pero en esa dirección de memoria almacenará el valor 12 y se ññamará num.
 
 

Operador de indirección (*)



El término puntero no significa otra cosa que una dirección de memoria. Ya hemos comentado que cuando declaramos una variable reservamos un lugar en memoria para dicha variable. Ese lugar reservado tiene una dirección de memoria y podemos obtener el valor de esa dirección mediante el operador &. Así mismo para obtener el valor del valor que contiene una dirección de memoria usamos el operador * sobre la variable de tipo puntero.

Vamos a figurarnos que hacemos las siguientes declaraciones:


int numero = 666;

int *puntero;

puntero = &numero;


Las direcciones de memoria de las variables numero y puntero son asignadas automáticamente por C. En este caso ha asignado las direcciones 0xbffff700 para puntero y 0xbffff716 para numero. Así mismo la variable numero tiene un valor de 666. En un principioo la variable puntero carece de valor.Esta varable sólo admite como valor valores que sean direcciones de memoria. En el momento que el hilo del programa ejecuta la siguiente linea puntero = &numero;queda asignada la dirección de memoria de numero a la variable puntero:


puntero = &numero;


Gráficamente podríamos interpretar esta última linea comno sigue:



           



Observa que las direcciones de memoria son diferentes en puntero y en numero. Observa también que la dirección de memoria de numero está contenida en puntero.

A partir de este momento es lo mismo referirnos se cumplen las siguientes igualdades:


*puntero == numero;

puntero == &numero;

&puntero != &numero;

Operadores de adición (+) y sustracción (-)



Los autores de este libro sobreentendemos que el lector sabe sumar y restar llegados a este punto. Por este motivo, suponemos que el entusiasmado y futuro programador que nos está leyendo no tendrá mayores problemas para comprender estos párrafos. Observa las siguientes líneas de código:

casera = 20;

En la primera línea asignamos el valor 10 a la variable vino, vino vale 10. En la segunda asignamos el valor 20 a la varible casera.

En la tercera línea asignamos el valor del resultado de la suma vino + casera (10 + 20) a la variable tinto_de_verano. A partir de este momento, la variable tinto_de_verano valdrá 30. Compruébalo con la función printf() en un programa.

Operador signo (-)

Al igual que en matemáticas si no precisamos el signo de la varible, esta variable mantendrá su signo original positivo o negativo. Si anteponemos el operador signo - a una variable, esta cambiará de signo. Observa: 


int numero_positivo;

int número_negativo = -12;

numero_positivo = -numero_negativo; /*esta sentencia asigna el número positivo 12 a la variable numero_positivo */


Como vemos, el programa asigna el número -12 a la variable numero_negativo. Luego antepone a numero_negativo el signo -, que es lo mismo que ejecutarr - ( -12), que como sabemos es lo mismo que el numero +12 o simplemente 12. Se ha asignado el número entero 12 a la variable numero_positivo.



Operador multiplicador ( * )




/*multiplica.c*/

#include <stdio.h>

main()

{

  int result_1;
  float result_2;

  result_1 = 3.3 * 3;
  result_2 = 3.3 *3;
  printf("result_1 = %d\n", result_1);
  printf("result_2 = %f\n",  result_2);

}







El operador multiplicador, multipica los dos números o variables situados a sus lados. Sin embargo, distinguiremos en asignar el resultado de la multiplicación de dos números reales a una variable entera (int) a asignar el resultado a una variable real (float). En el caso de nuestro anterior programa, aparentemente result_1 y result_2 deberían de tener el mismo valor, suponemos que 9.9. Esto será verdad mientras asignemos el resultado en una variable float. Si almacenamos el valor 9.9 en una variable entera int, en la variable  sólo quedará almacenada la parta entera del número, es decir hasta la coma decimal, a partir de este punto despreciará todo valor.si queremos que almacene el resultado con la parte decimal, deberemos de usar una varible tipo float.  Todas estas características, nos permitirán hacer programas más útiles e interesantes.
 
 



Operador de división (/) y operador módulo o de resto (%)





#include <stdio.h>

main()

   {
    int nuestro_dinero 5;
    int nosotros 3;
    int sobrante;
    int persona;

    persona = nuestro_dinero / nosotros;/*asigna el valor 1 a persona */

    sobrante = nuestro_dinero % nosotros;/*asigna el resto de la división 5/3 (2) a variable int sobrante.*/
    printf ("persona = %d\n", persona);
    printf ("sobrante = %d\n", sobrante);
   }



El operador / realiza la división entre los números situados a derecha e izquierda del operador y asigna el resultado en la varible indicada. Si queremos que el valor asignado sea un valor decimal, el resultado de la división ha de ser almacenado en uan varible float. Si queremos que sólo asigne la parte entera del resultado de la división, usaremos una variable int para almacenar el resultado. La salida del programa es:



persona = 1

sobrante = 2




El operador % realiza la división entre los números o variables situados al lado de aquel y ofrece como resultado el resto de la división.
