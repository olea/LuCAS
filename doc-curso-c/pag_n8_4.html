8.4.- Arrays de estructuras                          


Al igual que todos los tipos de variables en C, las estructuras pueden organizarse en arrays. Para poder visualizar la manera de hacerlo vamos a desarrollar un pequeño listín de fichas policiales que lo hizo el comisario Torrente de un pueblo llamado Cuatro Casas, para así poder tener controlados a los 2 delincuentes del citado pueblo. En cada estructura a parecerá el nombre, el apodo y las veces que ha estado detenido. Para declarar el array primero hemos de declarar la estructura y a continuación declararemos el array de la siguiente manera donde NUM es una constante definida con define:
 


struct ficha
        {
           char nombre[81];
           char alias[81];
           int n_veces;
        };
struct ficha fichas[NUM];



En este punto hemos declarado un array fichas de NUM estructuras de tipo ficha. Vemoas en el siguiente programa como introducimos datos y como los leemos datos del array fichas:
 


#include <stdio.h>


#define NUM 2

void main()
      {
        int i;
        char ch;
        struct ficha
                {
                 char nombre[81];
                 char alias[81];
                 int n_veces;
                 };

        struct ficha fichas[NUM];

        for (i=0; i<NUM; i++)

                 {

                  printf("Escribe el nombre del %dº fichado: ", i);

                  gets(fichas[i].nombre);

                  printf("Escribe el alias de %s: ", fichas[i].nombre);

                  gets(fichas[i].alias);

                  printf("Número de veces que ha estado detenido %s: ", fichas[i].alias);

                  scanf("%d", &fichas[i].n_veces);

                  while((ch = getchar()) == ' ' || ch == '\n')

                          break;

                  } 

        for(i=0; i<NUM; i++)

                  {

                   printf("fichas[%d].nombre = %s\n", i, fichas[i].nombre);

                   printf("fichas[%d].alias = %s\n", i, fichas[i].alias);

                   printf("fichas[%d].n_veces= %d\n", i, fichas[i].n_veces);

                  } 

         }

 



Cuando declaramos el array de estructuras mediante struct ficha fichas[NUM];¿la situación es exactamente igual a cuando declaramos un array de cualquier otro elemento. Eneste caso declaramos a fichas como el nombre de un array que contiene NUM estructuras. fichas es simplemente el nombre del array. Veamos la salida del programa.


Escribe el nombre del 1º fichado: Luis Sardino Acorralado


Escribe el alias de Luis Sardino Acorralado: Torpedo
Número de veces que ha estado detenido Torpedo: 1200
Escribe el nombre del 2º fichado: Pedro Mocho Malo
Escribe el alias de Pedro Mocho malo: Cabeza Trueno
Número de veces que ha estado detenido: 13
 
 

fichas[0].nombre = Luis Sardino Acorralado

fichas[0].alias = Torpedo

fichas[0].n_veces= 1200

fichas[1].nombre = Pedro Mocho Malo

fichas[1].alias = Cadeza Trueno

fichas[1].n_veces= 13



Para referirnos a cada uno de los elementos de cada estructura usamos como siempre el nombre de la estructura, el operador de pertenecia y la variable interna a la que nos queremos referir:
 

fichas[0].nombre 
fichas[1].alias 


Si declaráramos fichas[0].nombre[5], nos estariamos refiriendo al quinto elemento del array nombre que pertenece a la estructura fichas[0]. Nos estaríamos refiriendo al carácter 'S'.
