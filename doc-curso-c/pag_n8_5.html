8.5.- Punteros a estructuras                         


Como cualquier otro tipo de dato, si una estructura está alojada en una dirección de memoria, es lógico pensar que nos podamos referir a una estructura mediante un puntero. Podremos enviarlos y recibirlos a y de otras funciones, con la ventaja que ello supone. Cuando usamos un puntero a una estructura cambiamos el operador de pertenencia '.' por el operador apuntador '->'. Veamos como podemos acceder mediante un puntero a los datos de una estructura.
 


#include <stdio.h>

main()

        { 

          struct miestructura

                   {

                     int n_ladrillos; 

                     char t_construct[20]; 

                     int coste;

                   }; 
 

            struct miestructura estr=
                   {

                    1000,

                    "chabola y muro",

                    120000

                    };

           struct miestructura *punt = NULL;

           punt = &estr; 

/*comprobamos la dirección de la estructura y del primer elemento de la estructura */ 

           printf("&estr = %u &estr.n_ladrillos = %u\n", &estr, &estr.n_ladrillos);
          /* Usamos el operador -> */

           printf("punt->n_ladrillos = %d\n", punt->n_ladrillos);

           printf("punt->t_construct = %s\n", punt->t_construct);

           printf("punt->coste = %d\n", punt->coste);

 }



Hemos introducido los datos en la estructura estr en el propio programa aplicando los valores directamente, en la declaración de estr como variable de tipo miestructura. Cuando hacemos la declaración de str damos unos valores a estr.n_ladrillos, estr.t_construct y estr.coste. Todos estos valores están situados en memoria de una forma contigua.

Para poder determinar que un puntero apunte a una estructura, lo conseguimos declarando un puntero a una estructura de la forma siguiente:
 

struct miestructura *punt = NULL;


Le hacemos apuntar a NULL porque lo estamos inicializando y no queremos que contenga basura. Hemos declarado en este punto dos variables de tipo struct miestructura. Una es estr y la otra es un puntero a una estructura de tipo struct miestructura. Para que *punt apunte a estr hemos de asignar la dirección de estr a punt, y lo haremos de la manera siguiente:
 

punt = &estr;





Para referirnos a los elementos de estr, lo podemos hacer a través de estr o a traves del puntero punt. Para acceder a los miembros a través de estr ya hemos visto como hacerlo mediante el operador de pertenencia '.' mediante la fórmula:

estr.n_ladrillos
estr.t_consruccion
est.coste

Para acceder a los elementos del array apountado por *punt, hemos de hacerlo de la siguietne manera:
 

punt->n_ladrillos

punt->t_construct

punt->coste



Tambien podríamos insertar datos en estos elementos de la estructura mediante las funciones printf(), gets() y todas aquellas que hemos visto hasta ahora:
 

scanf("%d", &punt->n_ladrillos);
gets(punt->t_construct);



El tratamiento es idéntico que el dado a otras variables del mismo tipo.
En los arrays, el nombre del mismo era la dirección del primer elelemento del array, algo parecido pasa en las estructuras. En estas, la dirección de la estructura coincide con la dirección del primer elemento de la estructura. Para comprobarlo, hemos incluido en el programa la linea siguiente:
 


printf("&estr = %u &estr.n_ladrillos = %u\n", &estr, &estr.n_ladrillos);



La salida del programa en nuestro sistema es:
 


&estr = 3221223388 &estr.n_ladrillos = 3221223388

punt->n_ladrillos = 1000

punt->t_construct = chabola y muro

punt->coste = 120000


 

Observa que &estr y la dirección de memoria del primer elemento de la estructura &estr.n_ladrillos tienen el mismo valor.
