12.3.-Avanzar por los elementos de una lista.


Para avanzar por los elementos de una lista debemos de tener una referencia a la cabeza de la lista - cabeza -. Una vez con la referencia , deberemos declarar otro puntero a cabeza que sirva de indice y que es el que vamos a mover a lo largo de la lista.

struct datos
{
  int numero;
  struct datos *siguiente;
};
struct datos *cabeza;
struct datos *indice;

//-----

/*Código que crea nodos y los enlaza en la lista */

//-----

indice = cabeza;
while (indice != NULL)
{
          printf("indice->numero = %d\n", indice->numero);
          indice = indice->siguiente;
}
