9.7.- Funciónes generadoras de números aleatorios rand() y srand() 9.6.- Funciones de tiempo



Para poder generar números aleatorios usamos la función rand() que devuelve un número aleatorio comprendido en el rango entre 0 y el valor máximo para un int.

Sinopsis de rand()

#int rand(unsigned int semilla);


Para poder generar números aleatorios la función rand() necesita de una "semilla" o número a partir del cual se generan listas de números aleatorios. Si no especificamos el parámetro de rand(), esta tomará como valor de unsigned int semilla el valor 1. Los números aleatorios se generarán, pero con una secuencia idéntica. Para poder cambiar elvalor de unsigned int semilla usamos la función srand().

Sinopsis de srand()

void srand(unsigned int semilla);


Sería una buena opción aplicar como semilla el resultado de time() en nuestro sistema a fin de que la secuencia de números varíe en las diferentes ejecuciones del programa:

#include <stdio.h>

#include <stdlib.h>

#include <time.h>


main()

{

  long segundos;

  int i;

  long time_proc;


  srand(time(&segundos));

  printf("Hay va un aleatorio--> %d\n", rand());

  printf("Hay va otro aleatorio menor que 10 --> %d\n", rand() % 10);

  printf("Otro aleatorio menor que 1000 ----> %d\n", rand() % 1000);


/*Las siguientes líneas retrasan la ejecución de un mensaje por pantalla un segundo*/

  puts("\n\n10 sewgundos para el despegue");


  for (i=0; i< 10; i++)

    {

      do

         time_proc = clock();

      while((time_proc / CLOCKS_PER_SEC) < i);

      printf("%d segundos\n", 10-i);

    }

  printf("BBBBBRRRRRRRRRRRROOOOOOOOOOOOOUUUUUUUUUUMMMMMMMMMM\n");


}



Además este programa genera números aleatorios comprendidos entre rangos determinados. Esto es una buena pista para generar programas para la lotto. El bucle for() emite un mensje cambiante por pantalla cada segundo que pasa. Ejecuta y compila este programa para ver su comportamiento.
