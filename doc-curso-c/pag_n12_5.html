12.5.- Borrado de nodos en listas


Al igual que en el agregado de elementos, se puede borrar el elemento siguiente a indice o el mismo elemento apuntado por indice. El código para borrar el elemento siguiente al apuntado por indice es el siguiente:


struct datos *elemento_a_borrar;

elemento_a_borrar = indice->siguiente;
indice->siguiente = elemento_a_borrar->siguiente;
free (elemento_a_borrar);

Para borrar el elemento apuntado por indice haremos una copia del este nodo elemento apuntado código necesario sería:



struct datos *elemento_a_borrar;

elemento_a_borrar = indice->siguiente;
*indice = *elemento_a_borrar;
free (elemento_a_borrar);
