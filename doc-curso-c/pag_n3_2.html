3.2.- Tipos de datos en C



Los datos en los lenguajes de programación es de lo más aburrido, carecen de emoción y de velocidad y nos hacen bostezar, pero es inevitable. Hemos de darnos cuenta que en un principio en un programa pueden entrar todo tipo de datos, frases, letras, palabras, números grandes, pequeños y números decimales. El programador ha de saber distinguir entre estos tipos de datos con el fin de que el programa pueda identificarlo exactamene y declararlos como los ha interpretado, de esta manera, todo funcionará correctamente y el compilador no nos dará avisos de error, al menos en lo que a la definición de variables toca.

En el apartado de la declaración de variables, observamos varios aspectos nuevos: float, char además del ya conocido int.

Es fundamental que nuestro compilador conozca que tipo de datos vamos a escribir en el nombre de la variable que le hemos indicado.


int pags;
float altura;
float peso;
char simbolo='$';


Cuando ejecutemos el programa y este se encuentre estas declaraciones de variables hará lo siguiente:

int pags; Buscará un espacio en memoria y reservará espacio para asimilar el valor de esa variable. En este caso, a la variable pags no le hemos asimilado ningún valor. El espacio de memoria contendrá restos de bits de otros programas o datos perdidos, en todo caso en el momento de la declaración, el valor de pags es basura, no sirve para nada a nuestro programa. En este espacio de memoria sólo deberíamos almacenar un dato de tipo int. Y digo deberíamos porque si intentáramos escribir un dato distinto de int en el espacio de memoria asignado el programa lo aceptará, pero los resultados serán distorsionados y el resultado del programa será imprevisible.

float altura; En este caso el programa buscará espacio para guardar una variable de tipo float. Una variable de tipo float, es una variable que contiene un punto decimal.

char simbolo='$'; Este tipo de dato char, es nuevo para nosotros.. char declara una variable de tipo carácter. Un carácter es cualquier tipo de símbolo que contiene nuestro teclado. Este carácter, no tendrá aparentemente ningún tipo de valor, sólo significará para para nosotros un símbolo, no estando sujeto en un principio a ningún tipo de operación matemática.

Mediante la sentencia scanf("%d", &pags); en la linea 11, asignamos el valor entero captado por el teclado a la dirección de memoria de la variable pags. Observa que hemos especificado el tipo de dato como %d y cuando lo especificamos así, le indicamos al programa que en la dirección de memoria de la variable pags vamos a almacenar una variable de tipo entero.

scanf("%f", &altura); en la linea 13. Ahora debemos de captar por el teclado un valor float y asignarlo a la dirección de memoria de la variable altura. Observa que para almacenar un int antes insertábamos el identificador &d, sin embargo ahora se trata de un número float. En este caso utilizaremos el identificador %f. Que le indica al programa que en la dirección de la variable altura se va a introducir un número con coma decimal.

En las líneas 16 y 17 destacar que la sentencia printf() incluye un salto de línea. Si en el entrecomillado de una función printf() aparece un carácter de nueva linea ( \n ) el programa comenzará a escribir en la siguiente linea de texto a partir de dicho carácter de nueva linea.
