10.7.- Entrada y salida de cadenas de caracteres, fputs() y fgets()



Cuando el programa que estemos desarrollando pida a voces introducir cadenas de caracteres en un fichero previamente abierto, usaremos la función fputs() y para leer las cadenas de caracterres de un fichero usamos la función fgets().

    Sinopsis fputs()

    #include <stdio.h>
    int fputs(char *cadena, FILE *pf);

    Devuelve un valor distinto de 0 si todo ha ido satisfactoriamente (valor del último carácter escrito). Devuelve 0 en caso contrario.



Esta función copia una la cadena de caracteres *cadena en el archivo apuntado por *pf. Copia todo excepto el carácter fin de linea '\0'. Una vez hayamos terminado de escribir una cadena de caracteres podríamos añadir un carácter especial que lo podamos interpretar al leer el archivo. Por ejemplo podríamos por ejemplo posponer el carácter nueva línea '\n' y daría la impresión que estamos leyendo líneas de texto.



    Sinopsis fgets()

    #include <stdio.h>

    char *fgets(char *cadena, int n, FILE *pf)

    Devuelve un puntero al inicio de la cadena leida. Si a ocurrido un error o ha detectado el fin de fichero devuelve NULL.


La función fgets() lee datos del fichero apuntado por pf hasta encontrar un EOF ó el carácter nueva línea.
