8.2.- Inicialización de estructuras



Otra manera de inicializar aguila es:


struct animal

         {


          char nombre [20];

          int n_patas;

          char pelaje[10];

          int pico;

          int mamifero;

          char familia[20];

         }aguila;


De esta manera C crea una modelo de estructura animal y una variable de animal llamada aguila. Y a partir de aquí ya podríamos rellenar los datos internos de aguila.

En el ejemplo anterior hemos inicializado la estructura aguila con basura, con datos que no nos sirven y a continuación le hemos asigando valores. Tambien podemos inicializarla directamente con los datos que no interesan sin que dicha estructura contenga basura en ningún momento. Veamos como hacerlo:
 


void main()

        {

           struct animal

              {

               char nombre[20];

               int n_patas;

               char pelaje[10];

               int pico;

               int mamifero;

               char familia[20];
 

              };

/* Inicialización de aguila */

          struct animal aguila={"aguila", 2, "plumas", SI, NO, "rapaces"};
       }



Observa que hemos encerrado entre llaves los datos que va a contener la estructura aguila. El primer dato "aguila" se corresponde con el primer dato de la estructura aguila que es char nombre[20]. El segundo dato, 2, se corresponde con int n_patas. Si introdujéramos el orden de los datos aleatoriamente, el programa no lo entendería y obtendríamos un error en la compilación. Es fundamental que el orden secuencial de los datos que introducimos siga el orden de la estructura animal. Otra forma de hacerlo sería:
 


void main()

     {

       struct animal

            {

               char *nombre;

               int n_patas;

               char *pelaje;

               int pico;
               int mamifero;
               char *familia;
            }aguila={
                       "aguila",
                        2,
                        "plumas",
                        SI,
                        NO,
                        "rapaces"
                    };





Debido a que no queremos desperdiciar espacio de memoria (somo ahorradores), hemos sustituido las cadenas de caracteres por punteros a cadenas de caracteres, que nos permiten ahorrar unos bytes extra de memoria. El lector se preguntará por la importancia de declarar una estructura con valores basura y luego rellenarlos con los datos interesantes para nuestro programa o bien asignar a la estructura valores desde un principio. Por ejemplo, si en vez de una estructura animal fuera una estructura ventana (window) donde aparecen diversos datos sobre la ventana como dimensión, color, borde etc, al ejecutar la estructura ventana el usuario observaría una especie de operación de "refresco" de la ventana, una con los valores basura y otra con los valores asignados por nosotros. No obstante contemplamos ámbas maneras para que el apasionado lector capte la diferencia.

Incluso la intuición nos pude jugar malas pasadas al intentar hacer algo como lo que sigue:
 


animal aguila



Olvidándonos de fiugurar la notación struct delante de animal. La anterior declaración es válida en C++, no así en C. Para que podamos hacer este tipo de declaración debemos de declarar la estructura como sigue:
 


typedef struct animal

          {

             char nombre[20];

             int n_patas;

             char pelaje[10];

             int pico;

             int mamifero;

             char familia[20];
          };


A partir de este momento cada vez que escribamos en nuestro programa la palabra animal equivaldrá a struct animal, con lo que la declaración animal aguila equivaldría a declarar struct animal aguila y sería completamente correcta la declaración. No obstante, opinamos que esto sería intentar imitar otro lenguaje de programación que no es C. Dejamos a la elección del lector que escoja la mejor y más agradable manera de declara estructuras según sus criterios y necesidades.

Tambien podemos inicializar una variable de tipo struct animal en la propia definición de la estructura añadiendo al final de las llaves el nombre de la variable que usaremos. La forma sería:
 

struct animal

       {

         char nombre[20];

         int n_patas;

         char pelaje[10];

         int pico;

         int mamifero;

         char familia[20];
      }pato;





Esta declaración crea un modelo de estructura animal y un otro espacio de memoria para almacenar la estructura pato de tipo animal. A partir de este momento ya podremos añadir tantos valores a la variable pato como queramos.
