7.7.- Argumentos en la linea de comandos



En ciertas ocasiones necesitaremos pasar a nuestro programa ciertos argumentos para que los procese de una manera determinada. Por ejemplo podráimos imaginar un programa que contara el número de caracteres de la cadena que le pasamos por la linea de comandos. Si nuestro programa se llamara cuentaletras, la manera de ejecutarlo sería:

# cuentaletras palabreja

En la ejecución de este programa, cuentaletras sería el nombre del programa y hemos escogido como arguemento de el argumento de cuentaletras la cadena palabreja. Este programa tiene dos arguementos, uno el propio nombre del programa (cuentaletras) y un segundo argumento la cadena que le pasamos (palabreja). Veamos el funcionamiento de esto:

/***************argumentos.c***************/


#include <stdio.h>


void main(int argc, char *argv[])

{

  int i;

  printf("argc = %d\n", argc);

  printf("Has escrito los siguientes argumentos\n");


  for(i=0; i<argc; i++)

    {

       printf("argv[%d] = %s\n",i, argv[i]);

    }

}


Vamos a imaginarnos que compilamos este programa y le adjudicamos el nombre de argumentos.exe y lo ejecutamos como

# argumentos.exe segundo_arguemento tercer_argumento



el resultado sería:

argc = 3
has escrito los siguientes argumentos:
argv[0] = argumentos.exe
argv[1] = segundo_argumento
argv[2] = tercer_argumento


La línea que más nos interesa de este programa es

main(int argc, char *argv[])


Cuando ejecutamos el programa argumentos.exe, capturamos el número de argumentos del programa en int argc, de manera que si ejecutáramos un programa sin argumentos, es decir, con un sólo argumento, que es el propio nombre del programa, la variable int argc, adquiere el valor 1. En nuestro caso adquiere el valor 3 ya que le hemos pasado tres argumentos, el propio nombre del programa argumentos.exe y las cadenas de caracteres segundo_arguemento y tercer_argumento. Y las cadenas de caracteres que pasamos al programa como argumentos las capturamos en un array de punteros a cadenas de caracteres char *argv[]. Por convenio se usan como nombres de variables a argc y *argv[], pero puedes fabricar tu programa cambiando el nombre de las variables.

Veamos un programa que cuenta las letras del segundo arguemento de un programa:

#include <stdio.h>

void main(int argc, char *argv[])

{

   if (argc != 2)

      {

        printf("Modo de usar este programa: contador.exe <cadena de caracteres sin espacios en blanco>\n");

        exit(1);

      }

   printf("El número de caracteres de %s es %d\n", argv[1],    strlen(argv[1]));

}


En este programa nos interesa que usuario teclee el nombre del programa y una sóla cadena de caracteres. Todo lo demás no nos sirve, por tanto instalamos una condición if(argc!=2) para que el programa sólo se ejecute cuando tenga dos argumentos: El nombre del programa y la cadena de caracteres que vamos a medir, por tanto cada vez que el número de argumentos sean diferentes de 2, lanzamos un mensaje de error y sintaxis del programa y salimos del mismo. Para contar las letras del segundo arguemento del programa usamos la funcion strlen() sobre la cadena argv[1].

Para pasar una cadena de caracteres con espacios en blanco al programa usaremos comillas para ello:



# cuantaletras.exe "Superman ataca C"



Tambien podemos encontrar los argumentos de main de la manera:



void main(int argc, char **argv)





Que almacena los argumentos del programa en un array de punteros a punteros cadenas de caracteres.
Es muy correcto comenzar nuestros programas de la forma void main(int argc, char *argv[]) pese a que no vayamos a usar los datos recibidios como argumentos en línea de comanados.
