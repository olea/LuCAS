Bucle infinito de lectura en entrada estándar que hace:

- Leer la entrada estándar (una línea).
- Si EOF => FIN

- La entrada estándar contiene la URI parcial (fichero) solicitado.

- Comprueba si existe el fichero localmente
  Concatena PATH de copia local de LuCAS con la URI parcial.

- Si existe el fichero pedido =>

Mientras (no_tenga_URL_operativa)
HACER
  + Aplicar_algoritmo_de_selección_de_réplica (en función de turno actual,
porcentaje de carga, operatividad de la réplica).

  + Construir la URL final.

  + Si la URL funciona (petición HEAD):
    - devolver URL, salir del bucle MIENTRAS.
FIN-Mientras



- Nota: Cada N recorridos del round-robin, releer la base de datos y
actualizar la cache local.
