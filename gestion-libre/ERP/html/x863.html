3.13. Requisitos de portabilidad

Sistema operativo

El sistema debe poder instalarse en diferentes tipos de sistemas operativos sin que eso altere la definición del producto. Para este criterio es necesario contemplar que la solución pueda soportar la orientación del plan de sistemas de la compañía.

Migración de datos

El proceso de migración de datos tendrá como objetivo transferir los datos ubicados en las aplicaciones actuales al nuevo sistema. Este proceso se conforma de una serie de tareas que describen un flujo que determina el traspaso de la información desde el origen hasta su destino. Además, no se debe perder de vista que es una ocasión única para proporcionar integridad a la información que será distribuida a través de la nueva aplicación, evitando además trasladar cualquier incoherencia de información. El adjudicatario deberá encargarse la correcta migración de los datos actualmente utilizados al nuevo sistema.
