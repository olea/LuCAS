Capítulo 3. Requisitos específicos

3.1. Requisitos de interfaz externo

Cliente Web

El sistema deberá ofrecer la posibilidad de un acceso al sistema en entorno Web para la totalidad de sus transacciones.

Entorno de integración (API)

Disponer de un enterno de integración con aplicaciones externas mediante unas funciones de acceso remoto (API) permite desarrollar interfaces o utilizar información del ERP en conjunción con otros programas.
