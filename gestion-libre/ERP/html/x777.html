3.10. Requisitos de seguridad

Legislación vigente

Deberán señalarse las medidas de seguridad del sistema referidas en los artículos 9 y 12 de la Ley 15/ 1999, de 13 de diciembre, de Protección de datos de carácter personal.

El sistema deberá cumplir todas las exigencias de la normativa sobre protección de datos de carácter personal en el sentido que el sistema incorpore todas los archivos necesarios en los soportes adecuados para que la entidad pública pueda cumplir las obligaciones que le impone la Ley 15/ 1999.

Control de acceso

El sistema deberá tener un control de acceso para cada tipo usuarios. Los niveles de seguridad pueden variar en cuanto a las posibilidad de personalización de las autorizaciones de acceso a la información (formulario, informes, campos, menús, tareas, ...).

Otros

A nivel más técnico se valorará disponer de elementos de seguridad para la protección de la información de la base de datos o de las comunicaciones.

  • Control de autorizaciones de acceso al sistema

  • Encriptación de las comunicaciones y datos (Terminal / Servidor, Base de datos)

  • Control de acceso a la base de datos (permisos, registro de logs, ...)
