Otra de las funciones más importantes es la personalización del tipo
de teclado que poseemos en nuestras computadoras, si no se posee un
teclado Inglés, es necesario cargar el \emph{Centro de Control
KDE}\footnote{Anteriormente llamado ``KDE Control Center'', podemos
ver hasta que nivel de personalización se puede configurar el
escritorio}, entrar en la categoría \emph{Dispositivos de Entrada} y
seleccionar la opción \emph{Teclado Internacional} como en la figura
\ref{fig:CentroDeControlDispositivoDeEntrada-TecladoInternacional-General}

\figura{Selección del mapa de caracteres del teclado}{CentroDeControlDispositivoDeEntrada-TecladoInternacional-General}

Como se podrá ver, el mapa de caracteres por defecto que el KDE tiene
configurado inicialmente es para los teclados ingleses, si no se posee
un teclado de ese tipo, oprimir \boton{Eliminar} y luego
\boton{Añadir}, aparecerá un cuadro con una lista desplegable, donde
se puede seleccionar el mapa de caracteres para el teclado
adecuado. Con \boton{OK} los cambios tomarán efecto.
