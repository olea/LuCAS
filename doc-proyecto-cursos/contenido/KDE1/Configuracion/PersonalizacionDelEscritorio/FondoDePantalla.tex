\traductor{}

En la categoría \emph{Escritorio} del \emph{Centro de control KDE} se
tienen las diferentes opciones para configurar el aspecto y la
funcionalidad del entorno, la primera opción es \emph{Fondo}, y se
utiliza para configurar el color o imágen del fondo de los
escritorios.

\figura{Configuración del fondo de pantalla}{CentroDeControlEscritorio-Fondo}

Como se ve en la figura \ref{fig:CentroDeControlEscritorio-Fondo}, se
puede configurar cada escritorio por separado, o unificar la
configuración para todos seleccionado la opción \emph{Fondo
común}. En el sector \emph{Colores} se puede seleccionar un color de
fondo o utilizando la opción \emph{Dos colores}, se logra un
``degradé'' entre ambos. 

En la sección \emph{Tapiz} se encuentra una lista de las imágenes
que pueden utilizarse para el fondo. En la lista desplegable de
\emph{Disposición} existen varias opciones para la forma de mostrar
en pantalla dichos tapices.

Para que el fondo de pantalla cambie automáticamente, puede
activarse activar la opción \emph{Al azar}.

Si se activa la opción \emph{Fijar en el panel}, en la barra de
herramientas del KDE aparecerá un ícono especial para poder cambiar la
configuración del fondo directamente sin necesidad de entrar en el
Centro de Control KDE.
