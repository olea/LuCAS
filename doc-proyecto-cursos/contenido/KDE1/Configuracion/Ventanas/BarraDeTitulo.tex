Varios aspectos de la barra de título pueden modificarse, observando
la figura \ref{fig:CentroDeControlVentanas-BarraDeTitulo} se ve que
está dividido en cuatro secciones: el cuadro \emph{Alineación del
título} permite al usuario alinear el texto del título a la derecha,
centro o izquierda, debajo está el cuadro \emph{Aspecto} donde se
define el color de fondo de la barra de título.

Si se elige la opción de \emph{Pixmap}\footnote{Pixmap es un formato de
imágenes como lo es el GIF y el JPEG, la extensión que utiliza este
formato es ``xpm'', pero a veces se refieren a pixmaps como a imágenes
en general}, se activará a la derecha un cuadro donde se puede
seleccionar qué imagen de fondo de barra se utiliza cuando la ventana
está activa y en segundo plano.

\figura{Configuración de la barra de título de las ventanas}{CentroDeControlVentanas-BarraDeTitulo}

Debajo, la opción \emph{Título activo tiene borde con sombra} permite
darle -en caso de ser activada- un efecto en tres dimensiones a la
barra de título.

El cuadro siguiente, \emph{Acción del ratón}, asigna una acción
determinada al hacer ``doble-click'' en la barra de título, por
defecto la acción seleccionada es plegar la ventana de tal manera que
sólo se vea la barra de título, haciendo otro ``doble-click'' la
ventana se despliega.

Por último, el cuadro inferior \emph{Animación del título} posee una
barra de nivel para configurar la velocidad de animación de un título
cuando es demasiado largo para caber completo en la barra. Cuando esto
ocurre, el texto del título comienza a deslizarse de un lado al otro
para que el usuario pueda leerlo sin problemas.  

\textbf{NOTA:} Para mover una ventana de lugar, no es
imprescindible tener que posicionar el puntero del mouse en la barra
de título, hay veces que la barra no está dentro de los límites del
área de la pantalla, pero aún así se puede mover la ventana
presionando \boton{Alt} mientras se arrastra la ventana desde
cualquier sitio dentro de ella, no solo la barra de título.
