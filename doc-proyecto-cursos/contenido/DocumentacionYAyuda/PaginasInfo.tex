\porcion{Páginas \comando{Info}}
\autor{Guía de instalación oficial de Red Hat Linux}
\colaborador{\SGG}
\revisor{\LLC}
\traductor{}

Mientras \comando{man} es el formato de documentación más extendido,
\comando{info} es mucho más potente. Proporciona enlaces de hipertexto
para hacer mucho más sencilla la lectura de documentos grandes y muchas
ayudas para quien escribe la documentación.

Para leer la documentación de \comando{info}, use el programa \comando{info}
sin ningún argumento. Le presentará una lista de documentación disponible.
Si no puede encontrar algo, es probable que sea debido a que no tiene
instalado el paquete que incluye dicha documentación.

El sistema \comando{info} es un sistema basado en hipertexto. El texto
que aparezca resaltado es un enlace que apunta a más información. Las teclas
más importantes cuando maneja \comando{info} son:

\begin{description}
   \item [\boton{Tab}]
      para mover el cursor a un enlace
   \item[\boton{Intro}]
      para seguir un enlace
   \item[\boton{p}]
      le devolverá a la página previa
   \item[\boton{n}]
      le enviará a la página siguiente
   \item[\boton{u}]
      subirá un nivel de la documentación
   \item[\boton{q}]
      para abandonar info
\end{description}

La mejor manera de usar \comando{info} es leer la documentación
\comando{info} acerca de él. Si lee la primera página que \comando{info}
le presente, será capaz de comenzar a usarlo.
