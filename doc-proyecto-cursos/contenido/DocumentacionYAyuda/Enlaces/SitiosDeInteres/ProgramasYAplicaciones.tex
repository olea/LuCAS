%+++++++++++++++++++++++  Programas y aplicaciones ++++++++++++++++++++++

Hay multitud de programas y aplicaciones para GNU/Linux, la mejor forma de
encontrarlos es visitando las siguientes páginas:

\begin{description}

   \item[DistroWatch.com]
   	\sitio{http://www.distrowatch.com/}, Comparativas de las distribuciones
	                                     GNU/Linux más importantes. Un buen
					     punto de partida para conocer las
					     distintas distribuciones existentes

   \item[Freshmeat]
	\sitio{http://freshmeat.net}, Este sitio está dedicado a la recolección de
	                              Software Libre de todo tipo, dividido por
				      categorías

   \item[ibiblio]
   	\sitio{http://metalab.unc.edu}, Repositorio de software y distribuciones
	                                de GNU/Linux

   \item[Kernel.org]
   	\sitio{http://www.kernel.org}, Sitio principal para obtener el código
	                               fuente de Linux

   \item[LinuxISO]
   	\sitio{http://www.linuxiso.org}, Página donde podrás encontrar las imagenes
	                                 ISO de algunas distribuciones de GNU/Linux,
					 listas para ser grabadas

   \item[RmpFind]
   	\sitio{http://rpmfind.net}, Buscador de aplicaciones, la característica
	                            de este es que todos los programas están
				    empaquetados en RPM

   \item[SourceForge]
	\sitio{http://sourceforge.net/}, Servicios gratuitos para los desarrolladores
	                                 de Software Libre, en el que se podrán
					 encontrar aplicaciones de código abierto
					 de todo tipo

\end{description}
