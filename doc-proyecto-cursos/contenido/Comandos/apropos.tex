\porcion{El comando \comando{apropos}}
\autor{\NC}
\colaborador{}
\revisor{\LLC}
\traductor{}

Una herramienta útil para encotrar otras herramientas es el comando
\comando{apropos}. El comando \comando{whatis} muestra información en
base al nombre de un comando, posee su propia base de datos que relaciona
nombres con descripciones. \comando{apropos} busca en esa base de datos de
manera inversa, en base a una palabra, busca en las descripciones para
mostrar los comandos que concuerden.

\begin{ejemplo}

Por ejemplo si queremos utilidades relacionadas con la palabra \emph{jpeg}:

\begin{vscreen}
$ apropos jpeg
jpegtopnm            (1)  - convert JPEG/JFIF file to portable pixmap or graymap
pcdtoppm             (1)  - a program converting PhotoCD images to ppm or jpeg
ppmtojpeg            (1)  - convert portable pixmap into a JPEG/JFIF file

\end{vscreen}
%$

\end{ejemplo}
