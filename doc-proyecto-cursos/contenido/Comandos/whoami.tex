\porcion{El comando \comando{whoami}}
\autor{\LDP}
\colaborador{}
\revisor{\LLC}
\traductor{}

\'{E}ste es otro comando muy simple como \comando{pwd}. Su función
consiste en presentar en pantalla el nombre de usuario del usuario que
lo ejecuta. Ejemplo:

\begin{vscreen}
usuario@maquina:~/$ whoami
usuario
\end{vscreen}
%$
