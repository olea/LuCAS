\porcion{Pasarela de correos con \comando{sendmail}}
\autor{\LDP}
\revisor{\LLC}
\colaborador{}
\traductor{}

La función de \emph{pasarela de correos}\footnote{En inglés:
\emph{Mail relaying}} en \comando{sendmail} se utiliza para definir
los dominios que el servidor de correos tiene permitido dejar pasar.
