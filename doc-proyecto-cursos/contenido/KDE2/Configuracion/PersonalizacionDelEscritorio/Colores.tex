\porcion{Colores del entorno}

\autor{\LDP}
\colaborador{\JAB}
\revisor{}
\traductor{}

Los colores de los diferentes componentes de las ventanas pueden
cambiarse de forma tal de darle al entorno la apariencia que a uno
más le agrade. Esto se puede hacer en la sección de \emph{Colores} que
puede observarse en la figura
\ref{fig:CentroDeControlEscritorio-Colores}. Este cuadro se divide en
tres partes más una de visualización previa.

\figura{Configuración de colores en KDE}{fig:CentroDeControlEscritorio-Colores}
{width=6cm}
{KDE2/Configuracion/PersonalizacionDelEscritorio/CentroDeControlEscritorio-Colores}

Cada configuración de colores se puede guardar como un esquema
individual, en el cuadro \emph{Esquema de colores} se pueden ver
varios esquemas ya definidos, simulando los colores de varios
entornos. Pueden definirse nuevos esquemas, para esto se selecciona
cualquiera y se comienza a asignar un color para cada objeto
visual\footnote{Objeto Visual: se le llama a cada componente de una
ventana, un botón, un recuadro, la barra de título, etc.}, una vez
realizado esto, se presiona el \boton{Guardar Cambios..} para guardar
el nuevo esquema.En la visualización previa se puede ir controlando
los cambios de colores a medida que se realizan.

El cuadro \emph{Contraste} posee una barra de nivel que establece el
contraste entre los colores seleccionados.
