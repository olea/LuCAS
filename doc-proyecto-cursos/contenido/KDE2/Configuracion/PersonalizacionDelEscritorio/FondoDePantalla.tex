En la categoría \emph{LookNFeel} del \emph{Centro de control KDE} se
tienen las diferentes opciones para configurar el aspecto y la
funcionalidad del entorno, la opción \emph{Fondo} \footnote{También se puede acceder haciendo click con el boton derecho sobre el escritorio y seleccionanado la opcion ``Fondo''} se utiliza para configurar el color o imagen
del fondo de los escritorios.

\figura{Configuración del fondo de pantalla}{fig:CentroDeControlEscritorio-Fondo}{width=10cm}{KDE2/Configuracion/PersonalizacionDelEscritorio/CentroDeControlEscritorio-Fondo}

Como se ve en la figura \ref{fig:CentroDeControlEscritorio-Fondo}, se
puede configurar cada escritorio por separado, o unificar la
configuración para todos seleccionado la opción \emph{Fondo común}. En
la pagina \emph{Fondo} se selecciona el tipo de fondo a utilizar,
estos pueden ser:
\begin{itemize}
	\item{Un color único de fondo.}
	\item{Varios tipos de gradientes entre dos colores.}
	\item{Un programa de fondo.}
	\item{Una página web como fondo de pantalla.}
\end{itemize}
En la página \emph{Papel Tapiz} configuramos si queremos poner como
fondo de pantalla un archivo de imagen y de que manera mostrarlo,
además permite seleccionar múltiples papeles tapices que se van
cambiando cada un intervalo de tiempo determinado,para realizar esto
seleccionar la opción ``Múltiples Papeles Tapiz'' y presionar
\boton{Configuración múltiple...}, para agregar las imágenes a mostrar
y configurar el intervalo de tiempo.  Por ultimo se le puede dar un
toque mas ``artístico'' al fondo de nuestro escritorio, para esto
seleccionamos la página \emph{Avanzado} donde podemos mezclar el color
de fondo seleccionado en la primer página con el papel tapiz elegido,
existen distintos tipos de mezclas con diferentes grados de intensidad.
Con \boton{Aplicar} guardamos los cambios realizados.
