Esta sección permite personalizar las clases de tipografías que se
mostrarán en los diferentes sectores del escritorio y objetos
visuales, en la figura \ref{fig:CentroDeControlEscritorio-Fuentes}
puede observarse que existen varias categorías diferentes para
configurar, cada una de estas categorías se les puede asignar un
estilo de tipografía, su tamaño y características, teniendo siempre la
posibilidad de previsualizar los cambios.
