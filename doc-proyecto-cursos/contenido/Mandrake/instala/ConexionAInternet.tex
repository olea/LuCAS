La otra opción es la conexión usando módem, la cual es sin temor
a dudas, la más popular. La configuración de la conexión pregunta
si intenta encontrar el módem y si no tiene éxito, preguntará
el puerto serie al cual está conectado. Tal como se explicó 
anteriormente, este dispositivo será {\it /dev/ttySx}, muy 
posiblemente el \verb+/dev/ttyS3+ correspondiente al COM4 en DOS/Windows.
Las características de la conexión se piden a continuación en
una caja de diálogo (figura ~\ref{fig_configura_modem}).

\begin{itemize}
\item{{\bf Nombre de la conexión} para identificar esta conexión}
\item{{\bf Número de teléfono} a cual se va a llamar}
\item{{\bf ID de conexión} Su identificación de usuario }
\item{{\bf Contraseña} Su palabra clave}
\item{{\bf Autenticación} El tipo de autenticación. Por defecto es PAP}
\item{{\bf Nombre del dominio} La extensión al nombre, ej: (skina.com.co).
		 No requerido}
\item{{\bf Primer servidor DNS} Servidor de Nombres primario.
		 No requerido pero recomendado}
\item{{\bf Segundo servidor DNS} Servidor de Nombres secundario.
		 No requerido pero recomendado}
\end{itemize}
