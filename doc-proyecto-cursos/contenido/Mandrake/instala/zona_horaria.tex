En este momento, el proceso de instalación muestra 
una pantalla con el resumen de la configuración hecha
hasta ahora y algunos valores tomados por defecto. 
Para configurar cualquiera de ellos basta con 
presionar con el ratón el botón asociado.

El resumen presenta los siguientes valores: 

\begin{itemize}
\item{{\bf Ratón}: Serie 2 botones genérico}
\item{{\bf Teclado}: Español }
\item{{\bf Zona Horaria}: Europa/Madrid}
\item{{\bf Impresora}: Sin Impresora}
\end{itemize}


El primero a cambiar es la zona horaria. En la caja de 
diálogo que se muestra está
una lista jerárquica en forma de árbol de Continente/Ciudad 
que rige las diferentes zonas
horarias en el mundo. Si no se encuentra la ciudad, se escoge
una que tenga la misma hora que su ubicación, por ejemplo
{\sf América/Bogotá} para toda Colombia
(figura~\ref{fig_zona_horaria}).

Esta selección es muy importante para las correcciones horarias de
los países con estaciones. A la pregunta {\sf El reloj de su
computador usa GMT?} se contestará negativamente.
