A continuación, el sistema presenta la pantalla de selección 
del Sistema de Archivos. \verb+DrakX+ usa la herramienta 
gráfica \verb+diskdrake+ para realizar esta labor. La gráfica 
muestra en colores la forma en que se encuentra
particionado el disco duro: rojo para {\it ext2}, verde para
{\it Swap}, azul para {\it FAT} (Windows) y blanco para vacío. 
Al seleccionar una partición en la parte
inferior se muestran las posibles operaciones con ella (izquierda)
y la información de su estado actual (derecha). En la parte inferior se
encuentra el botón de [{\sf Asignación automática}], el cual
realiza la tarea por el usuario y está recomendado por el autor. Aquí
se prefiere realizarlo de forma manual ya que se quiere controlar 
el tamaño de las particiones. 
Todos los cambios que se hagan no serán guardados hasta que
se seleccione [{\sf Hecho }]. Si en cualquier momento se descubre que se
ha cometido un error grave se podrá recuperar el estado inicial
presionando en la derecha la opción  [{\sf Mas}]${\rightarrow}$[{\sf Recargar}]
y éste reiniciará el proceso. Una vez que se haya seleccionado
[{\sf Hecho}] y aceptado los cambios, no existe posibilidad de recuperar el
estado inicial (figura~\ref{fig_Disk_Drake}).

\figura
{Editando particiones con {\tt Diskdrake}}
{fig_Disk_Drake}
{width=11cm}{Mandrake/configura/install/pan_particiones.png}

Si se han seguido las instrucciones de esta guía, existirá una primera
partición para DOS/Windows (Azul) y un espacio vacío a continuación. Si este
es el caso, se selecciona el espacio vacío y a la izquierda la opción
[{\sf Crear}] para añadir una nueva partición, \verb+diskdrake+ 
presentará una caja de diálogo con la siguiente información
(ver figura~\ref{fig_adicione_particion}):
\begin{itemize}
\item{Sector de comienzo: el cual se deja en el valor por defecto, a menos que se quiera la partición en un lugar físico particular en el disco.}
\item{El tamaño de la partición: en donde se introduce el valor en 
{\it Megabytes} de la partición. Este valor puede ser escrito o usar la barra de {\it desplazamiento}.}
\item{ Tipo de sistema de archivos: {\it Linux native} para los archivos de Linux y {\it Linux swap} para la partición de intercambio}
\item{ Punto de montaje: El directorio donde será asignada la partición. Si 
es una partición dedicada a todo el sistema de archivos de Linux
habrá que introducir el símbolo ``/'', que significa el directorio raíz. La partición de intercambio no tiene punto de montaje.}
\item{Preferencia: {\it Extendida} o {\it Primaria} si se desea una partición lógica o una primaria}
\end{itemize}

\figura
{Agregar una partición}
{fig_adicione_particion}
{width=4cm}{Mandrake/configura/install/pan_nueva_particion.png}

Entonces se crea primero la partición de intercambio como 
estaba planeado: se 
pulsa sobre la partición vacía, y sobre [{\sf Crear}], no se introduce 
ningún punto de montaje. El tamaño será lo que se desee, seguramente 
128 Mbytes como mínimo y el tipo será {\sf Linux swap}.
Como se menciona anteriormente, el tamaño de 128 Mbytes para la 
partición de intercambio es solamente tentativo, dependiendo del 
uso del sistema y de la memoria RAM que tenga. Se presiona [{\sf Aceptar}]
para continuar. Y se observa la nueva partición verde.
Se presiona nuevamente lo que resta del disco en blanco y 
se procede a [{\sf Crear}]. El punto de montaje es ``/'' en
donde irán todos los archivos de Linux. De tamaño se toma todo el
espacio restante y por supuesto el tipo será {\sf Linux Native}.
Después de [{\sf Aceptar}] se tiene completamente copado el disco
con tres particiones y se encuentra listo para continuar 
seleccionando [{\sf Hecho}]. 


Una vez creadas las particiones, el proceso de instalación continúa y la
siguiente pantalla pregunta qué particiones formateará. Estas 
particiones serán las que contengan el sistema de archivos de Linux.
Es posible que se hayan creado algunas de las particiones como
lógicas y no primarias. En este caso hay que recordar que éstas
se llamarán, por ejemplo {\it /dev/hda5} o {\it /dev/hda6}.
La lista de particiones muestra el nombre y el punto de montaje,
y se seleccionan ambas presionando el botón a su izquierda. 
Se escoge también {\sf Verificar el disco en busca
de bloques malos} por precaución (figura ~\ref{fig_formateo}) .
