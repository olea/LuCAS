Una vez ha comenzado la instalación es cuestión de seguir cada
paso que indica el computador (figura ~\ref{fig_flujo_inst}). 
Hay dos posibilidades de interfaz: una gráfica (por defecto) 
y otra en texto, pero ambas son equivalentes. En este texto sólo 
se presentarán las pantallas gráficas en X11.
Si se tiene problemas para la instalación gráfica, ya sea porque
su tarjeta no cumple con el estándar VESA 1.0 o por cualquier 
otra razón, el instalador entrará por defecto al modo texto. Se 
puede intentar un modo gráfico de baja resolución, presionando 
en el inicio
[{\sf F1}], para más opciones y escribiendo en la pantalla de 
presentación ``vgalo'' o la simple consola de texto escribiendo 
``text'' y así se comenzará la instalación en el modo 
escogido. 

\begin{vscreen}
boot: text
\end{vscreen}

La primera pantalla muestra \verb+DrakX+: la instalación
gráfica de \distribucion Linux. 
A la izquierda se muestra la secuencia de tareas a seguir: en verde
las ya realizadas, en naranja la actual y en rojo las que no se
han ejecutado. Estas tareas pueden ser seleccionadas de forma 
independiente presionando con el ratón el botón con la estrella.
Al final de esta lista, se pueden observar tres\footnote{Sólo tres
cuando se usa baja resolución VGA} o cuatro pequeñas 
barras de color azul claro, verde, gris y azul rey,
las cuales permiten escoger los colores a usar en la instalación (Temas).
En la parte inferior derecha están las instrucciones y la 
descripción de las diferentes opciones de cada proceso (esta puede
ser ampliada presionando el botón con la interrogación). 

A continuación se pregunta por el lenguaje que se desea usar
durante la instalación. Por defecto se sugiere el
Inglés, pero se permite realizarla en 
Castellano ({\sf Español}) el cual es ofrecido con varios regionalismos.
(figura~\ref{fig_escoger_leng}).

\figura
{Escogiendo un lenguaje para la instalación}
{fig_escoger_leng}
{width=11cm}{Mandrake/configura/install/pan_lang.png}
                        
Una vez se escoge el idioma se procede a escoger 
el estilo que seguirá la instalación, y se tienen las 
siguientes opciones:

\begin{itemize}
\item{{\bf Recomendada}: para principiantes que nunca han instalado Linux}
\item{{\bf Experto}: Donde todas las decisiones las toma el usuario}
\end{itemize}

En esta guía se utiliza la instalación de {\sf Experto}, ya que
permite afinar más detalladamente diversos aspectos de la
instalación. La otra opción realiza la mayoría de
operaciones automáticamente, incluyendo la selección
de muchos dispositivos y paquetes.


El instalador intentará en este momento detectar
dispositivos SCSI y pedirá que se le indique
si existe algún otro dispositivo en el sistema.
Si no se sabe lo que es un dispositivo SCSI, entonces 
seguramente no se tiene ninguno instalado y 
puede contestar {\sf no}. \verb+DrakX+ puede instalar el soporte 
para la mayoría de dispositivos de forma automática. En caso de
que no lo logre, preguntará por los parámetros del dispositivo: 
posiblemente la dirección de entrada/salida y la interrupción 
que utiliza. No se darán más detalles
ya que un PC corriente no tiene interfaces SCSI.

Ahora se procede con la selección de dispositivos de 
entrada y salida, iniciando por la elección del ratón.
Existe una buena cantidad de ratones que pueden usarse
con GNJ/Linux, incluyendo los nuevos {\it Scroll Mouse},
pero seguramente el que se posee es un
{\sf Genérico serial de 2 botones} conector de 9 pines 
{\it rectangular} o un ratón {\sf Genérico PS/2}, de conector 
redondo. Si se escoge un ratón serial, es necesario 
señalar el puerto en el cual está conectado. Éste se detecta
automáticamente y es normalmente el COM1 {\it /dev/ttyS0} 
(figura~\ref{fig_escoge_raton}).

Si el ratón posee únicamente dos botones, el sistema escoge 
automáticamente la emulación de tres botones,
ya que la mayoría de aplicaciones utilizan más de dos botones.
Con esta emulación al presionar los dos botones al mismo tiempo, hacen la
vez de un tercero. 


\figura 
{Seleccionando ratón}
{fig_escoge_raton}
{width=11cm}{Mandrake/configura/install/pan_raton.png}

En la siguiente pantalla se deberá indicar cuál es el tipo de teclado
que se pretende usar. Se busca el modelo de teclado que corresponde al
que se encuentra usando. Para este caso se escogerá el teclado
{\sf Español} (figura~\ref{fig_teclado_espaniol}). También se da 
otra opción de configuración: {\sf Estadounidense (con acentos)}, 
pero si se desea uno diferente a los ofrecidos por defecto, 
se presiona [{\sf Más}] para ver las otras opciones de configuración 
de teclado. 

\figura
{Seleccionando modelo y distribución del teclado}
{fig_teclado_espaniol}
{width=11cm}{Mandrake/configura/install/pan_teclado.png}

Una vez se ha escogido el teclado, se pregunta por el nivel de seguridad 
del sistema. Existen tres opciones de un extremo a otro. La seguridad
{\sf Baja} está orientada a estaciones de trabajo en redes locales, 
mientras que la opción {\sf Alta} es recomendada para
servidores, i.e. máquinas con muchos usuarios.
Se escoge Media/Alta, aunque la última, un usuario anterior de
Linux/Unix puede encontrarla un poco restrictiva.
