Aun cuando se ha configurado una impresora durante la
instalación con \verb+DrakX+, no se está exento de
afinar su configuración,
añadir una nueva o simplemente cambiar de impresora.

\iconoconsola
Como se menciona al inicio del presente capítulo,
el centro de control de \distribucion, \verb+DrakConf+ 
dispone de un conjunto de botones para diferentes 
tareas de administración, entre ellas incluye un botón 
de configuración de Impresoras presionando
[{\sf Sistema}]${\rightarrow}$[{\sf Printer}] el cual llama a la 
misma herramienta utilizada en el momento de la instalación:
 \verb+printerdrake+. Por esto la Sección
~\ref{sec_impresora} puede ser seguida paso a paso para instalar
las impresoras. Se escoge su versión en texto para las
ilustraciones.

Al iniciar \verb+printerdrake+ muestra las colas instaladas
(figura ~\ref{fig_prdrk_colas}) y un botón de adición [{\sf Add}]. 
Si no se ha instalado una impresora previamente el sistema inicia
por defecto añadiendo una cola. 

\figura
{{\tt Printerdrake}: Colas de impresión}
{fig_prdrk_colas}
{width=6cm}{Mandrake/configura/impresora/colas.png}

Se presenta ahora la posibilidad de escoger la conexión a la
impresora: remotas usando LPR (Unix) o SMB (MS-Windows),
o local por puerto paralelo. Se escogerá una impresora local.

\figura
{{\tt Printerdrake}: Tipos de conexión de impresoras}
{fig_prdrk_where}
{width=9cm}{Mandrake/configura/impresora/where.png}


\verb+Printerdrake+ tratará de detectar el dispositivo donde se 
encuentra conectada la impresora, que por defecto es el primer 
puerto paralelo conocido como {\it /dev/lp0}.

Aparecerá entonces la ventana de descripción con toda
la información de la impresora: el Nombre: lp por 
defecto\footnote{ Herencia
del formato de LPD}, la descripción y la ubicación. 

\figura
{{\tt Printerdrake}: Descripción de impresora}
{fig_prdrk_descr}
{width=9cm}{Mandrake/configura/impresora/descrip.png}


La instalación presentará una lista de impresoras
(figura ~\ref{fig_prdrk_modelo}). Se selecciona
la correspondiente o la que más se parezca al modelo.
Una vez seleccionada se pregunta si desea hacer una
prueba, la cual se recomienda.

\figura
{{\tt Printerdrake}: Modelos de impresoras}
{fig_prdrk_modelo}
{width=9cm}{Mandrake/configura/impresora/modelo.png}
                         

Una vez configurada la impresora se presentará nuevamente
la pantalla de colas y puede presionar hecho [{\bf Done}].
configuración pulsa en [{\sf OK}].
