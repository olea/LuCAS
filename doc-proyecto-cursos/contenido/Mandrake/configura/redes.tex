\traductor{}

Una vez se tiene la tarjeta de red configurada, se
tiene un dispositivo denominado {\it eth0} y ahora
se le asignan los valores que ya se mencionaron
en la instalación en la sección ~\ref{sec_inst_red}.

La herramienta utilizada durante la instalación está
disponible también para configuración: \verb+draknet+,
en donde se pueden asignar todos los valores de
dirección IP, {\it gateway} y {\it proxy}.
