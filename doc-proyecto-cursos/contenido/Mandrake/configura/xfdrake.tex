Este programa es una producción original de {\bf Red Hat}. Ha
evolucionado a \verb+XFdrake+ en Mandrake.
Este programa es en modo consola con ventanas o gráfico
en otras versiones (figura ~\ref{fig_xconfigurator}).
Permite configurar la tarjeta gráfica, el monitor, resolución, 
etc. Tiene además una excelente capacidad de auto-detección
de dispositivos.
