\traductor{}

Después de la impresión, el espacio en disco es el
recurso más difundido dentro de las redes locales.
Para compartir recursos, GNU/Linux hereda de UNIX un
estándar conocido como NFS ({\it Network File System}) 
originado por {\bf Sun} en los años 70. El otro 
protocolo usado por GNU/Linux es SMB, implementado
por {\bf Samba} para compartir tanto impresión como 
archivos con MS-Windows.
