La inclusión de nuevo hardware o de hardware que no fue
apropiadamente instalado es una de las labores que eran
bastante temidas hace algún tiempo. \distribucion provee varias
herramientas que permiten la detección y configuración
de estos equipos.

\iconowindows

La herramienta más poderosa que posee \distribucion para la
configuración de {\it hardware} es \verb+HardDrake+
el cual integra tres niveles de programas:


\figura
{{\tt HardDrake}: Herramienta de administración de dispositivos}
{fig_harddrake}
{width=9cm}{Mandrake/configura/harddrake.png}


\begin{itemize}
\item{{\sf Librería Detect}: Es la máquina de auto-detección reuniendo
                        varias herramientas como isapnptools y SuperProbe}
\item{{\sf Harddrake}: La interfaz gráfica que combina la detección con 
                        la configuración}
\item{{\sf Harddrake Wizard}: Es una herramienta de configuración genérica
                        y es el reemplazo de \verb+Soundrake+ y \verb+Etherdrake+}
\end{itemize}


Para iniciar la herramienta, en una consola se escribe 
\verb+harddrake+ o desde el botón de configuración de hardware 
en \verb+DrakConf+.
Esto puede tomar algo de tiempo mientras realiza su tarea.
Una vez se activa, muestra un administrador gráfico 
de dispositivos (figura ~\ref{fig_harddrake}).

Se señala el dispositivo que se desea configurar y se 
ejecuta la herramienta particular para él, por ejemplo
la herramienta de configuración de sonido i.e. \verb+sound-wizard+
(figura ~\ref{fig_harddrake_sound}) que aparece cuando
se presiona [{\sf Corra herramienta de configuración  }] 
({\it Run configuration Tool}).

\figura
{{\tt HardDrake}: Sound-wizard}
{fig_harddrake_sound}
{width=9cm}{Mandrake/configura/harddrake-soundwizard.png}

En esta ventana se pueden cambiar los datos de interrupciones (IRQ) o
demás, cuestión que no es muy buena idea ya que éstos han sido pre-seleccionados
como consecuencia de la detección. Cuando se presiona [{\sf OK}] se 
ejecuta una prueba de tres sonidos: 8 bits, 16 bits y MIDI. Si se oyeron
correctamente, se presiona nuevamente [{\sf OK}] para confirmar
la configuración. Más adelante se presentan otras formas de configurar
el sonido.
